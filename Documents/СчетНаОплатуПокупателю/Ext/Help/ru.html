Документ «Счет на оплату покупателю»

Документ «Счет на оплату» предназначен для выписки счетов покупателям. Документ не является обязательным, он лишь содержит платежные реквизиты продавца, по которым покупатель осуществляет наличный или безналичный перевод денежных средств за перечисленные товары или услуги.

Документ «Счет на оплату покупателю» предназначен для ввода и хранения информации, проводки и записи в регистры документ не формирует. По счету на оплату можно производить контроль оплаты и отгрузки.

На основании документа «Счет на оплату покупателю» могут быть оформлены следующие документы:

  • Документ «Платежное поручение входящее». Документ предназначен для регистрации факта оплаты по счету в случае безналичной оплаты покупателем;
  • Документ «Платежный ордер поступление денежных средств». Документ предназначен для регистрации факта оплаты по счету в случае безналичной оплаты покупателем;
  • Документ «Приходный кассовый ордер». Документ предназначен для регистрации факта оплаты по счету в случае наличной оплаты покупателем в кассу организации;
  • Документ «Оплаты от покупателя платежными картами». Документ предназначен для регистрации факта оплаты по счету в случае, если покупатель расплачивается с помощью платежной карты;
  • Документ «Реализация ТМЗ и услуг». Документ предназначен для отгрузки товаров на основании выписанного счета покупателю. При оформлении документа реализации на основании счета, документ реализации заполняется всеми товарами, выписанными в счете на закладке «ТМЗ», без учета ранее оформленных документов реализации;
  • Документ «Передача ОС». Документ предназначен для отгрузки реализации основных средств организации на основании выписанного счета покупателю. При оформлении документа реализации на основании счета, документа реализации заполняется всеми основными средствами, указанными в счета на закладке «ОС», без учета ранее оформленных документов передачи;
  • Документ «Акт об оказании производственных услуг». Документ предназначен для отражения операций по оказанию услуг. При оформлении документа реализации на основании счета, документ реализации заполняется всеми номенклатурными позициями (услугами), указанными в счете на закладке «Услуги», без учета ранее оформленных документов реализации.

При создании нового документа поля «Структурная единица» и «Склад» заполняются автоматически значениями, указанными в настройках пользователя в группе «Основные значения для подстановки в документы и справочники». В поле «Банковский счет» автоматически устанавливается счет, который отмечен как основной для указанной структурной единицы.

Формирование счета на оплату может производиться как по произвольным ценам, так и по цене договора контрагента. Автоматически тип цены и валюта документа устанавливаются в соответствии с аналогичными данными в договоре контрагента. Изменить тип цены, валюту, учет НДС можно в форме «Цены и валюта», которая открывается по гиперссылке формы документа. Эти данные могут быть скорректированы вручную и в самом документе.

В табличных частях документа указывается список товаров и услуг (с количеством, ценами, сумма и т.д.), продаваемых покупателю.

В поле «Комментарий» можно ввести любую дополнительную информацию. Весь текст комментария можно просмотреть или отредактировать, нажав на кнопку «Выбрать».

Для документа «Счет на оплату покупателю» предусмотрена возможность формирования печатной формы «Счет на оплату», которую можно распечатать и передать покупателю.
