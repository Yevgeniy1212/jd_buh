Документ «Возврат ТМЗ поставщику»

Документ «Возврат ТМЗ поставщику» служит для отражения операций по возврату ранее приобретенных основных средств, услуг, товаров поставщику. Документ формирует проводки, как и документ «Поступление ТМЗ и услуг», но в «обратном направлении». На основании этого документа могут быть введены документы:

  • «Счет-фактура (выданный)»;
  • «Приходный кассовый ордер»;
  • «Счет-фактура полученный»;
  • «Платежное поручение (входящее)»;
  • «Платежный ордер (поступление денежных средств)»;
  • «Сторнирование».

В документе «Возврат ТМЗ поставщику» заполняются следующие реквизиты:

  • «Структурная единица» – покупатель;
  • «Контрагент» – поставщик;
  • «Договор» – договор, по которому будет осуществляться возврат;
  • «Документ расчетов» – реквизит заполняется, если используется учет по расчетным документам и выбран договор с типом «По расчетным документам»;
  • «Склад» – указывается склад, с которого будут возвращены ТМЗ.

Табличная часть «ТМЗ» - служит для отражения операций по возврату ТМЗ. Она состоит из:

  • Номенклатура – товары, возвращаемые продавцу;
  • Количество – указывается количество возвращаемого товара;
  • Цена – цена за единицу товара, по которой производится возврат;
  • Сумма – указывается итоговая сумма;
  • % НДС – указывается ставка НДС;
  • Сумма НДС – заполняется сумма НДС;
  • Всего – сумма, на которую осуществлен возврат;
  • Счет учета (БУ) – счет учета запасов, с которого будет списана себестоимость возвращаемой номенклатуры;
  • Счет учета НДС– счет учета для отражения суммы НДС по возврату для бухгалтерских проводок;
  • Счет доходов (БУ) – счет учета для отражения суммы дохода от реализации;
  • Вид оборота по НДС - аналитика счета НДС для регистра «Корректировка НДС к возмещению»;
  • Вид поступления для НДС - аналитика счета НДС для регистра «Корректировка НДС к возмещению».
  • Источники происхождения - служит для указания источников происхождения у товаров. Заполняется, если в конфигурации используется учет товаров по источникам происхождения

На закладке Счета учета расчетов заполняются:

  • Счет учета расчетов - счет, на который будут отражены проводки по учету взаиморасчетов;
  • Счет учета возвратов – счет, с которым будут корреспондировать проводки по возврату себестоимости и НДС по возвращаемым ТМЗ. Если счет указан, будут выполнены зачеты неоплаченной части возврата на счет учета расчетов с контрагентами. В случае, если счет не заполнен проводки по возврату себестоимости и НДС будут корреспондировать со счетом учета расчетов с контрагентом
  • Счет списания разницы в себестоимости (БУ) - при возврате товаров поставщику нередко возникает ситуация, когда себестоимость товара в табличной части (сумма без НДС) не совпадает с себестоимостью этого же товара по учетным данным организации. В этом случае доход/убыток от такой разницы (сумма без НДС - себестоимость по учетным данным) будет отнесен на счет списания себестоимости.
  • Аналитика списания разницы (БУ) - аналитика счета списания разницы.

Документ формирует проводки по бухгалтерскому учету. При проведении формируются проводки по списанию себестоимости ТМЗ со счета учета, указанные в табличной части «ТМЗ», формируется кредиторская задолженность на счетах учета расчетов с контрагентами и счетах учета расчетов по возвратам, кроме того формируются проводки по отражению НДС по приобретению, а также формируются проводки по отражению дохода/убытка от разницы в себестоимости.

Одновременно с проводками по бухгалтерскому учету НДС производятся записи в регистре накопления «Корректировка НДС к возмещению» на суммы НДС, указанные в документе.

Принятие НДС к зачету

При "стандартной" схеме, т.е. когда налоговый счет-фактура получен вместе с первичными документами.

При отражении операций возврата от поставщика будут сформированы бухгалтерские проводки по кредиту счета 1421 "Налог на добавленную стоимость" и движения по регистру накопления "Корректировка НДС к возмещению".

Отражение "нестандартной" схемы отнесения НДС в зачет.

Ситуация по отложенному принятию НДС к зачету возникает в случае, когда налоговый счет-фактура получен позднее первичных документов. В этом случае устанавливается признак "Отложить принятие НДС к зачету" в форме "Цена и валюта" по ссылке "Тип цен".
Будет сформирована бухгалтерская проводка по кредиту счета 1422 "Налог на добавленную стоимость (отложенное принятие к зачету)". Движения по регистру накопления "Корректировка НДС к возмещению" будут отсутствовать.
