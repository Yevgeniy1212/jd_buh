Документ "Выработка спецодежды и инвентаря"

Конфигурация "1С-Рейтинг: Учет спецодежды и инвентаря".

Документ «Выработка спецодежды и инвентаря» (раздел «Спецодежда и инвентарь» - «Выработка и амортизация» - «Выработка спецодежды и инвентаря») предназначен для отражения фактического количества оборотов материалов в эксплуатации. В дальнейшем, данные по количеству оборотов используются для расчета амортизации спецодежды и инвентаря с производственным способом погашения стоимости.

В документе указывается подразделение организации, в котором числится в эксплуатации материал с производственным способом погашения стоимости.

В табличной части «Материалы» указывается:

  • Номенклатура – спецодежда и инвентарь, по которым регистрируется выработка. Значение реквизита выбирается из справочника «Номенклатура» и является обязательным для заполнения.
  • Характеристика – характеристика спецодежды и инвентаря, по которым регистрируется выработка.
  • Единица измерения- единица измерения спецодежды и инвентаря.
  • Коэффициент - коэффициент перевода в базовую единицу измерения.
  • Количество - количество спецодежды и инвентаря, по которым регистрируется выработка. Для каждой единицы материала, указанной в данной строке, количество оборотов будет одинаковым.
  • Документ передачи – первичный документ передачи материалов в эксплуатацию.
  • Сотрудник\Физ. лицо - сотрудник\ физическое лицо, (если учет ведется в разрезе аналитики по физическим лицам), ответственный за материал. Значение реквизита выбирается из справочника «Физические лица», либо «Сотрудники» и является обязательным для заполнения только для списания из эксплуатации.
  • Назначение использования - назначение использования спецодежды и инвентаря. Значение реквизита выбирается из справочника «Назначения использования» и является обязательным для заполнения.
  • Количество оборотов - количество оборотов, выполненное каждой единицей указанного материала. Например, если количество инвентаря - 10, а количество оборотов - 3, значит каждый из этих  10, совершил 3 оборота в эксплуатации.

Проведение документа

При проведении документа формируются записи по регистру оперативного учета «Выработка материалов в эксплуатации». Движения по бухгалтерскому и налоговому учету  при проведении документа не формируются.
