Документ «Установка соответствия счетов БУ и НУ»

Документ предназначен для добавления, изменения и удаления правил соответствия счетов БУ и НУ.

В табличной части предусмотрено заполнение следующих реквизитов:

  • «Счет БУ»- выбирается счет бухгалтерского учета, для которого задается правило соответствия;
  • «Счет корр БУ» - выбирается счет бухгалтерского учета, корреспондирующийся с тем счетом, для которого задается правило соответствия (необязательный);
  • «Комментарий» - произвольный комментарий;
  • «Субконто (БУ)» - указывается конкретное значение (значения) субконто счета бухгалтерского учета, если именно для этого значения (набора значений) задается особое правило соответствия (необязательно);
  • «Счет НУ» - выбирается счет налогового учета, для которого задается правило соответствия;
  • «Вид затрат (НУ)» - если одно из субконто имеет справочник «Статьи затрат» или «Доходы», то можно указать необходимое значение (необязательно);
  • «Субконто (НУ)» - указывается конкретное значение (значения) субконто счета налогового учета, если именно для этого значения (набора значений) задается данное правило соответствия (необязательно).

По кнопке «Заполнить» можно автоматически заполнить табличную часть правилами соответствия по умолчанию.

При проведении документа заполняется регистр сведений «Соответствие счетов БУ и НУ».
