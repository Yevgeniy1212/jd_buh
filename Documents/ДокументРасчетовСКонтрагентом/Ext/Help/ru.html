Документ «Документ расчетов с контрагентом (ручной учет)»

Вспомогательный документ, который не делает никаких проводок, предназначен для ведения ручного учета по расчетным документам. Реквизиты необходимые для заполнения:

  • «Структурная единица»;
  • «Контрагент»;
  • «Договор».
