Документ «Ввод начальных остатков»

Документ предназначен для ввода всех входящих остатков в программу.

Ввод остатков осуществляется в разрезе различных разделов учета.

Раздел учета выбирается в реквизите «Раздел учета».

Разделы учета сгруппированы в соответствии с планом счетов бухгалтерского учета.

Для ввода данных, для которых не выделен отдельный раздел учета, следует выбрать раздел «Прочие счета бухгалтерского учета».

В зависимости от выбранного раздела будет меняться состав данных, которые необходимо ввести и форма документа.

На верхней панели документа размещена кнопка «Скрыть справку». При нажатии не нее, в правой части документа выходит окно, в котором содержится справочная информация о выбранном разделе учета.

В документе вводятся остатки по бухгалтерскому и налоговому учету, и (или) специальным регистрам. Для выбора режима ввода остатков (ввод остатков по бухгалтерскому учету, по налоговому учету, по специальным регистрам или по всем видам учета) нужно нажать кнопку «Режим ввода остатков» в верхней командной панели.

В правой части документа размещена краткая информация с описанием общих принципов работы при вводе остатков по соответствующему разделу учета. В документе предусмотрена возможность автоматического заполнения данных налогового учета по данным бухгалтерского учета при помощи кнопки «Заполнить по остаткам БУ» (меню «Заполнить» - «Заполнить по остаткам БУ») и «Заполнить реквизиты НУ по данным БУ» (меню «Заполнить» - «Заполнить реквизиты НУ по данным БУ»).

Остатки вводятся в корреспонденции со вспомогательным счетом 000. После окончания ввода остатков на вспомогательном счете 000 сальдо на дату ввода остатков должно получиться равным нулю.

После окончания ввода входящих остатков рекомендуется произвести контроль введенных данных, сформировав отчет по входящим остаткам.
