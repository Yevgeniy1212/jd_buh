Документ "Списание спецодежды и инвентаря"

Конфигурация "1С-Рейтинг: Учет спецодежды и инвентаря".

Документ «Списание спецодежды и инвентаря»предназначен как для списания новой, бывшей в употреблении спецодежды и инвентаря, находящихся на складе, так и для списания спецодежды и инвентаря, находящихся в эксплуатации (раздел «Спецодежда и инвентарь» - «Эксплуатация спецодежды» - «Списание спецодежды и инвентаря»). Данная операция позволяет списать спецодежду и инвентарь и отнести остаточную стоимость на счет затрат указанный в назначении использования  для способа погашения расходов, либо на счет затрат указанный на закладке «Счета учета затрат» документа  списания.

Необходимо указать:

  • Состояние материалов - определяет, какая спецодежда и инвентарь должны списываться - находящиеся в эксплуатации, либо находящиеся на складе.
  • Склад - место хранения, с которого проводится списание новой и б\у спецодежды и инвентаря. Значение реквизита выбирается из справочника «Склады», доступно только для состояний «Находящиеся на складе (новая)», «Находящаяся нна складе (б\у)».
  • Подразделение организации - подразделение организации, в котором эксплуатировались спецодежда и инвентарь на момент списания. Значение реквизита выбирается из справочника «Подразделения организаций», доступно только для состояния «Находящиеся в эксплуатации».
  • Способ списания расходов - выбор варианта отражения операции списания по бух. учету:
    • По назначению материалов - списание стоимости по бух. учету производится на основании способа погашения стоимости, указанного в назначении использования для спецодежды и инвентаря.
    • По параметрам документа - списание стоимости проводится на счет и аналитику затрат, указанный на закладке «Счета учета затрат». Данный способ рекомендуется применять для отражения операции списания спецодежды и инвентаря из эксплуатации при увольнении сотрудников\физических лиц, либо при порче материалов.
  • Причина списания - причина, по которой происходит списание спецодежды и инвентаря. Выбирается из справочника «Причины списания».

В табличной части «Материалы» указывается:

  • Номенклатура - спецодежда и инвентарь подлежащий списанию.
  • Характеристика – характеристика списываемой спецодежды и инвентаря.
  • Количество - количество списываемой спецодежды и инвентаря.
  • Единица измерения - единица измерения спецодежды и инвентаря.
  • Коэффициент- коэффициент перевода в базовую единицу измерения. 
  • Документ передачи – первичный документ передачи материалов в эксплуатацию.
  • Сотрудник\Физ. лицо - сотрудник\ физическое лицо, (если учет ведется в разрезе аналитики по физическим лицам), с которого списывается находящийся в эксплуатации материал. Значение реквизита выбирается из справочника «Физические лица», либо «Сотрудники» и является обязательным для заполнения только для списания из эксплуатации.
  • Назначение использования - назначение использования спецодежды и инвентаря в эксплуатации. Значение реквизита выбирается из справочника «Назначения использования».

Имеется возможность заполнить списываемые товары автоматически при помощи команд подменю «Заполнение»:

  • материалами с истекшим сроком – табличная часть заполняется материалами, у которых истек срок эксплуатации;
  • по остаткам - табличная часть заполняется по всем остаткам на выбранном складе\подразделении;
  • по документу передачи, по складу, по сотруднику – позволяет выбрать из списка соответствующее значение (документ передачи, склад, сотрудника) и заполнить табличную часть всеми материалами, числящимися с указанным признаком;
  • по сроку эксплуатации – позволяет выбрать остаточный срок эксплуатации, после чего материалы, остаточный срок эксплуатации которых равен выбранному сроку или меньше его, попадут в табличную часть;
  • заполнить по документу передачи – предварительно очищает табличную часть и заполняет ее материалами из выбранного документа передачи;
  • добавить из документа передачи – добавляет в табличную часть материалы из выбранного документа передачи;
  • заполнить по документу перемещения – предварительно очищает табличную часть и заполняет ее материалами из выбранного документа перемещения;
  • добавить из документа передачи – добавляет в табличную часть материалы из выбранного документа перемещения.

На вкладке «Комиссия» можно указать состав комиссии, участвующей в списании материалов, для дальнейшего отображения в печатных формах.

Формируемые печатные формы:

  • 3-6 Акт списания материалов;
  • Списание материалов – вывод на печать списка списываемых материалов;
  • Акт списания (количественный);
  • Акт списания (количественно-суммовой).

Проведение документа

При проведении документа формируются записи по следующим регистрам оперативного учета:

  • «Материалы на складе» – формируются движения по списанию спецодежды и инвентаря с выбранного в документе склада (если используется складской учет) в разрезе характеристик (если ведется учет по характеристикам);
  • «Материалы в эксплуатации» – формируются движения по списанию спецодежды и инвентаря с физических лиц в разрезе характеристик (если ведется учет по характеристикам).

При способе списания расходов «по назначению материалов по бухгалтерскому учету формируются движения по счету учета затрат, указанному в назначении использования материалов.

Если способ списания расходов указан «По параметрам документа», то сформируются движения по счету учета, указанному в назначении использования спецодежды и инвентаря для отражения амортизации за текущий месяц. Это позволит определить остаточную стоимость спецодежды и инвентаря на момент списания. Затем будут сформированы движения по списанию спецодежды и инвентаря и отнесению остаточной стоимости на счет затрат, указанный на вкладке «Счета учета затрат».

Если на момент списания спецодежды и инвентаря исчерпаны остаточная стоимость и срок использования, то по бухгалтерскому учету проводится списание только количества материала с нулевой стоимостью.

В налоговом  учете формируются проводки, только при установленном флаге «Учет КПН». Движения будут сформированы по счетам налогового учета (НУ) согласно установленным соответствиям счетам бухгалтерского учета (БУ).

Анализ данных

После проведения документа, с помощью отчетов конфигурации, можно проанализировать следующие данные:

  • с помощью отчета «Ведомость по спецодежде и инвентарю на складе» можно проанализировать количество списанной со склада спецодежды и инвентаря в различных разрезах (по характеристикам, складам, и т.д.);
  • с помощью отчета «Ведомость по спецодежде и инвентарю в эксплуатации» можно проанализировать количество и сумму (только для товаров, стоимость которых списывается линейным способом) списанной из эксплуатации спецодежды и инвентаря в различных разрезах (по характеристикам, физическим лицам, назначениям использований, и т.д.).

Проанализировать данные по бухгалтерскому учету  можно с помощью типовых отчетов «Оборотно-сальдовая ведомость по счету», «Карточка счета».
