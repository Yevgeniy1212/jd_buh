Документ «Установка цен номенклатуры»

Документ «Установка цен номенклатуры» предназначен для установки цен товаров, материалов, продукции и услуг. При проведении документа информация о ценах записывается в регистр сведений «Цены номенклатуры» по каждой позиции номенклатуры.

Цены можно вводить для каждой организации, зарегистрированной в информационной базе. При этом для одной и той же номенклатурной позиции, но для разных организаций цены могут различаться.

При вводе документа в шапке обязательно указывается «Тип цен», для которого задаются значения цен номенклатуры. В табличную часть документа вводится список номенклатурных позиций (материалы, продукция, услуги), для которых устанавливаются цены и валюта. Установленная документом цена действует до момента установки новой цены.

Табличная часть документа может быть заполнена автоматически, при помощи кнопки «Заполнить». Если в выпадающем меню выбран пункт «Заполнить по ...», то табличная часть документа очищается и заполняется заново отобранными позициями. Если выбран пункт «Добавить по …», то в табличную часть документа добавляются выбранные позиции.

Варианты автоматического заполнения табличной части документа:

  • «Заполнить по номенклатуре» – табличная часть заполняется всеми позициями из справочника «Номенклатура». Цены заполняются действующими значениями из регистра сведений «Цены номенклатуры». Если для номенклатуры нет записи в регистре сведений, то значение цены будет пустым;
  • «Заполнить по ценам номенклатуры» – табличная часть заполняется только теми номенклатурными позициями, для которых ранее уже были установлены цены для данного типа цен на дату оформления документа. Заполнение осуществляется из регистра сведений «Цены номенклатуры»;
  • «Добавить по ценам номенклатуры» – к уже введенным строкам табличной части добавляются строки по номенклатурным позициям, для которых ранее уже были установлены цены данного типа цен на дату оформления документа;
  • «Обновить по ценам номенклатуры» – в табличной части документа устанавливаются текущие значения цен для тех номенклатурных позиций, для которых они заданы в регистре сведений «Цены номенклатуры»;
  • «Заполнить по себестоимости»;
  • «Добавить по себестоимости»;
  • «Обновить по себестоимости»;
  • «Заполнить по поступлению» – табличная часть документа заполняется списком номенклатуры и ценами, которые определены в документе «Поступление ТМЗ и услуг»;
  • «Добавить по поступлению» – к уже введенным строкам табличной части добавляется список номенклатуры и цены из выбранного документа «Поступление ТМЗ и услуг».

Документ может быть оформлен как свободный документ, так и на основании документа «Поступление ТМЗ и услуг». Если документ «Установка цен номенклатуры» оформляется на основании документа поступления, его табличная часть автоматически заполняется данными из документа-основания. Цены в этом случае устанавливаются в соответствии с ценами, указанными в документе основания.

Если документ оформляется как свободный документ, его табличные части заполняются построчным вводом (при помощи кнопки «Добавить») или подбором (при помощи кнопки «Подбор») товаров из справочников «Номенклатура».

Кнопка «Изменить» предназначена для вызова обработки «Обработка табличной части Товары» для группового изменения табличной части «ТМЗ и услуги».

Закладка «Дополнительно» предназначена для ввода дополнительной информации по документу:

  • «Ответственный». Из справочника «Пользователи» указывается ответственный за документ пользователь. При создании нового документа заполняется автоматически данными текущего пользователя.
  • «Документ-основание». Указывается документ, на основании которого был создан или заполнен документ «Установка цен номенклатуры». При изменении значения документа-основания предоставляется возможность перезаполнения данных документа данными документа-основания.

Для документа «Установка цен номенклатуры» предусмотрена возможность формирования печатной формы «Перечень цен».
