Документ «Изменение параметров начисления амортизации НМА»

Документ «Изменение параметров начисления амортизации НМА» предназначен для отражения изменений в регистре сведений о нематериальных активах.

В документ вносятся новые сведения о нематериальных активах (новый срок полезного использования в случае его изменения, стоимость для начисления амортизации или объем продукции (работ)).

Для более быстрого заполнения табличной части документа можно воспользоваться кнопкой «Подбор».

После формирования списка нематериальных активов, параметры амортизации которых, нужно изменить, необходимо воспользоваться кнопкой «Заполнить» для заполнения табличной части информацией о нематериальных активах по данным учета, существующим на момент введения документа.

При проведении документа эти сведения записываются в регистр и используются документом «Закрытие месяца» при начислении амортизации в месяце, следующем за месяцем изменения параметров.

Данный документ не формирует никаких проводок, он лишь изменяет значение соответствующих ресурсов регистра сведений, хранящую информацию о нематериальных активах.

Для изменения остальных характеристик учета нематериальных активов (например - способа отражения расходов по амортизации) используется другой документ - «Изменение способов отражения расходов по амортизации НМА».
