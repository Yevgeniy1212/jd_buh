Документ "Установка норм выдачи спецодежды и инвентаря"

Конфигурация "1С-Рейтинг: Учет спецодежды и инвентаря".

Документ предназначен для регистрации информации по нормам выдачи спецодежды и инвентаря работникам организации. Документ доступен для заполнения только при установленном флаге «Ведется контроль выдачи спецодежды  по нормам» в настройках параметров учета спецодежды и инвентаря.

Для вида операции доступны следующие значения:

  • По должностям - задание норм по должностям в целом.
  • По сотрудникам\физическим лицам (если учет ведется в разрезе аналитики по физическим лицам) - задание норм индивидуально для каждого сотрудника\физического лица.

Каждому виду операции соответствует своя табличная часть. Например, при указании операции «По должностям» будет доступна для заполнения табличная часть «Должности», при указании операции «По сотрудникам\физическим лицам», будет доступна для заполнения табличная часть «Сотрудники\Физические лица».

Табличные части документа могут быть заполнены автоматически с помощью кнопки "Заполнить", при нажатии на которую предлагается выбрать способ заполнения:

  • Заполнить нормами выдачи - этот способ применяется при заполнении текущего документа, данными документов установки норм выдачи материалов за предшествующий период (например, в случае изменения нормативов выдачи). В этом случае табличные части «Сотрудники\Физические лица», «Должности» и «Нормы выдачи» заполнятся базисными значениями, которые были установлены ранее и хранятся в учетном регистре "Нормы выдачи спецодежды и инвентаря" на дату заполнения документа.  
  • Подбор - при этом способе заполнения открывается специализированная форма подбора, с помощью которой можно выбрать сотрудников\физические лица, либо должности организации.

В табличной части  «Сотрудники\Физические лица» отражаются данные по сотрудникам\физическим лицам (если учет ведется в разрезе аналитики по физическим лицам), которым передаются спецодежда и инвентарь. Значение выбирается из справочника «Сотрудники», либо справочника «Физические лица» и является обязательным для заполнения.

В табличной части «Должности» отражаются данные по должностям организации. Значение выбирается из справочника «Должности организации».

В табличной части «Нормы выдачи» отражаются следующие данные:

  • Номенклатура - спецодежда или инвентарь, по которым устанавливаются нормы выдачи. Значение выбирается из справочника «Номенклатура». У номенклатуры должен быть установлен признак, что она является спецодеждой\инвентарем.
  • Назначение использования - назначение использования спецодежды и инвентаря в эксплуатации. Значение реквизита выбирается из справочника «Назначения использования».
  • Количество - количество нормы выдачи спецодежды и инвентаря.

Проведение документа

При проведении документа формируются записи по регистру оперативного учета:

  • Нормы выдачи спецодежды и инвентаря – формируются движения значений норм выдачи спецодежды и инвентаря в разрезе организации, сотрудника\физического лица, номенклатуры и назначения использования.

Анализ данных

После проведения документа, с помощью отчета «Нормы выдачи спецодежды и инвентаря» можно проанализировать данные по установленным нормам выдачи и фактически выданным спецодежде и инвентарю в различных разрезах (по сотрудникам\физическим лицам, назначению использования).
