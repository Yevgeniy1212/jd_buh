Документ "Заявление на предоставление вычетов ИПН"

Документ предназначен для регистрации права предоставления налоговых вычетов физическому лицу по индивидуальному подоходному налогу в соответствии со статьей 343 Налогового Кодекса.

В шапке документа заполняются следующие сведения:

  • Организация
  • Структурное подразделение
  • Сотрудник
  • Месяц
  • Действует по

Применение вычетов будет осуществляться по периоду указанному в поле Месяц. Начало действия и прекращения применения вычетов устанавливается пользователем. Если поле Действует по не будет заполнено, то вычеты будут применяться без ограничения. Вычеты, для которых предусмотрен график платежей, применяются согласно данных графика и периода его действия.

В табличной части документа заполняются сведения о предоставленных видах вычетов, а именно:

  • Вид вычета
  • Установление признака – Предоставлять вычет
  • Документ, подтверждающий право на вычет.

Для вычетов, которые предоставляются по периодическим платежам, предусмотрен ввод графика платежей, по гиперссылке График.
В графике указывается размер и период действия вычета.

Обязательным условием для проведения документа является установка признака наличие заявления.
