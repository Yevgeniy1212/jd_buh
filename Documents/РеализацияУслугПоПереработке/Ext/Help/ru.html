Документ «Реализация услуг по переработке»

Документ предназначен для отражения в бухгалтерском и налоговом учете операции оказания услуг по выпуску продукции из давальческого сырья и отражения отчета об использованных при выпуске продукции давальческих материалах.

В документе заполняются следующие реквизиты:

  • «Структурная единица» – организация или структурная единица организации, осуществляющая переработку сырья заказчика;
  • «Выписан» - способ выписки Акта выполненных работ. Может принимать значение: В бумажном виде, На портале ИС ЭСФ или На портале гос.закупок;
  • «Дата подписания» - значение заполняется пользователем самостоятельно после подписания акта получателем, если способ выписки указан На портале ИС ЭСФ или На портале гос.закупок. Если способ выписики указан: В бумажном виде, дата подставляется автоматически после записи и будет равна дате создания документа.
  • Признак Учитывать КПН влияет на отражение данных документа по налоговому учету. Если в учетной политике по налоговому учету для указанной организации снят признак Плательщик налога на прибыль, то признак Учитывать КПН и все счета налогового учета будут не доступны;
  • «Контрагент» - заказчик переработки сырья. Значение реквизита выбирается из справочника «Контрагенты». Реквизит является обязательным для заполнения;
  • «Договор» - договор с заказчиком переработки сырья;
  • «Документ расчетов» - указывается расчетный документ. Указание расчетного документа возможно, если в выбранном договоре в строке «Взаиморасчеты ведутся» установлено – «по расчетным документам».

На закладке «Продукция (услуги по переработке)» указывается продукция, выпущенная из давальческого сырья, а также услуги по выпуску продукции. На закладке указываются:

  • «Номенклатура» – продукция, выпущенная из давальческого сырья. Значение выбирается из справочника «Номенклатура» и является обязательным для заполнения;
  • «Содержание услуги по переработке» - дополнительная информация о выпущенной продукции или оказанной услуге по переработке;
  • «Спецификация» – спецификация продукции, если она заведена для данной продукции. Значение реквизита выбирается из справочника «Спецификации номенклатуры». Данная информация позволяет автоматически рассчитывать состав материалов, необходимых для производства указанной продукции;
  • «Количество» - количество выпущенной продукции;
  • «Цена» – цена выпущенной продукции;
  • «Сумма» – произведение количества и цены;
  • «% НДС» – ставка НДС;
  • «Сумма НДС» – сумма НДС;
  • «Всего» – сумма с учетом НДС;
  • «Стоимость (плановая)» - плановая цена 1 единицы произведенной продукции;
  • «Сумма (плановая)» - плановая стоимость произведенной продукции;
  • «Счет учета (БУ)» - счет бухгалтерского учета по производству продукции из давальческого сырья (например, 8120);
  • «Счет НДС» - счет учета для отражения НДС;
  • «Вид реализации» – выбирается соответствующий вид реализации из справочника «Виды реализации ТМЗ (работ, услуг)»;
  • «Счет доходов (БУ)» - счет бухгалтерского учета доходов от оказания услуг;
  • «Субконто доходов (БУ)»- объекты аналитики для целей бухгалтерского учета доходов;
  • «Счет списания себестоимости (БУ)» - счет бухгалтерского учета расходов по оказанию услуг;
  • «Субконто списания (БУ)» - объект аналитики для целей бухгалтерского учета списания себестоимости;
  • «Счет учета (НУ)» - счет учета по налоговому учету;
  • «Счет доходов (НУ)» - счет учета доходов от оказания услуг по налоговому учету;
  • «Субконто доходов (НУ)» - объекты аналитики для целей учета доходов по налоговому учету;
  • «Счет списания себестоимости (НУ)» - счет учета расходов по оказанию услуг по налоговому учету;
  • «Субконто списания (БУ)»- объекты аналитики для целей списания себестоимости по налоговому учету;

Заполнение табличной части также возможно с помощью кнопки «Подбор» и кнопки «Заполнить». Заполнение по кнопке «Заполнить» может производиться по отчету производства за смену, который будет указан из предложенного списка или по остаткам счета «Производство из давальческого сырья».

На закладке «Материалы заказчика» указываются материалы, принятые в переработку, которые были использованы при выпуске продукции:

  • «Номенклатура» - материал, принятый в переработку. Значение реквизита выбирается из справочника «Номенклатура» и является обязательным для заполнения;
  • Количество - количество материала;
  • «Счет учета (БУ)» - счет бухгалтерского учета материала, принятого в переработку и переданного в производство. Значение реквизита выбирается из плана счетов бухгалтерского учета.

Предусмотрено заполнение данной табличной части по кнопке «Заполнить». Заполнение производится давальческими материалами, которые были переданы в производство и учтены на счете 9022 «Материалы, переданные в производство» или состав материалов можно рассчитать по спецификациям и продукции, указанным на закладке «Продукция (услуги по переработке)».

На закладке «Счета учета расчетов» в документе заполняются «Счет учета расчетов с контрагентом» и «Счет учета расчетов по авансам». Эти счета заполняются в соответствии с заданными в регистре сведений «Счета учета расчетов с контрагентами» значениями по умолчанию для каждого контрагента. При необходимости пользователь может изменить корреспонденцию счетов.

На закладке «Дополнительно» отражаются:

  • «Ответственный»- из справочника «Пользователи» указывается ответственный за документ пользователь. При создании нового документа заполняется автоматически данными текущего пользователя;
  • «Документ – основание»– документ, на основании которого могут быть заполнены данные в документе «Реализация услуг по переработке»;
  • «Реквизиты грузополучателя» – указывается наименование грузополучателя, выбираемое из справочника «Контрагенты», адрес поставки товаров;
  • «Дополнительная информация» – содержит поля, в которых возможно указать банковский счет организации, отчетный период и перечень документации.

По данным табличной части «Продукция (услуги по переработке)» отражается реализация услуг по переработке по плановым ценам и списание готовой продукции, произведенной из материалов заказчика со счета 8120 по плановым ценам.

По данным табличной части «Материалы заказчика» отражается списание материалов заказчика со счета 9022, как использованных на производство продукции для заказчика, отчет по которым принят заказчиком.
