Прекращение предоставления вычетов ИПН

Документ предназначен для прекращения всех действующих вычетов по ИПН для сотрудника. При создании документа автоматически выводится список всех предоставляемых вычетов сотруднику.
В документе не предусмотрено удаление или изменение вида вычета.

В шапке документа заполняются следующие сведения:

  • Организация
  • Структурное подразделение
  • Сотрудник
  • Месяц.

В поле Месяц указывается тот период, с которого будет прекращено предоставление вычетов.

В табличной части документа заполняются сведения о предоставленных видах вычетов:

  • Вид вычета
  • Действует по
  • Документ, подтверждающий право на вычет.
