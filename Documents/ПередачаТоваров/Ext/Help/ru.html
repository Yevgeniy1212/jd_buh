Документ «Передача ТМЗ»

Документ «Передача ТМЗ» предназначен для отражения операции по передаче собственных материалов переработчику для выпуска продукции и передачи заказчику продукции, выпущенной из давальческого сырья.

Документом можно оформить следующие виды операции:

  • «Передача сырья в переработку»;
  • «Передача продукции заказчику».

Вид операции определяется в поле «Вид операции». При выборе вида операции автоматически меняется внешний вид документа (изменяется состав элементов шапки документа и граф табличных частей). Вид операции желательно указывать при вводе нового документа. Но в любой момент времени его можно изменить (даже для проведенного документа) и конфигурация автоматически пересчитает все данные в информационной базе.

При создании нового документа поля «Структурная единица» и «Склад» заполняются автоматически значениями, указанными в настройках пользователя в группе «Основные значения для подстановки в документы и справочники».

Признак Учитывать КПН влияет на отражение данных документа по налоговому учету. Если в учетной политике по налоговому учету для указанной организации снят признак Организация является плательщиком корпоративного налога, то признак Учитывать КПН и все счета налогового учета будут не доступны.

На закладке «ТМЗ» указывается продукция, выпущенная из давальческого сырья, которая передается заказчику или сырье, передаваемое переработчику. Табличная часть заполняется построчным вводом (при помощи кнопки «Добавить») или подбором (при помощи кнопки «Подбор») товаров из справочника «Номенклатура». Автоматическое заполнение табличной части осуществляется при помощи кнопки «Заполнить». При автоматическом заполнении доступны два варианта:

  • Заполнить «По продукции и спецификации». При выборе этого режима заполнения открывается форма подбора номенклатуры, в которой появляется дополнительная колонка с указанием спецификации номенклатуры. При переносе данных в документ переносится не выбранная номенклатурная позиция (готовая продукция), а материалы (сырье), используемые для ее изготовления.
  • Заполнить «По остаткам 8120». Данный режим заполнения доступен только для вида операции «Передача продукции заказчику». Табличная часть заполняется всеми остатками номенклатурных позиций по счету 8120 «Полуфабрикаты собственного производства», которые числятся на остатках в организации на указанном складе.

При выборе номенклатурной позиции в табличную часть «ТМЗ» счета учета и счета передачи заполняются автоматически из регистра сведений «Счета учета номенклатуры».

В табличной части "Источники происхождения" указываются источники происхождения продукции или сырья. Заполняется, если в конфигурации используется учет товаров по источникам происхождения.

Закладка «Дополнительно» предназначена для ввода дополнительной информации по документу:

  • Ответственный. Из справочника «Пользователи» указывается ответственный за документ пользователь. При создании нового документа заполняется автоматически данными текущего пользователя.
  • Доверенность (№, дата). Выданной. Указываются реквизиты доверенности физического лица, получившего товарно-материальные запасы.

Документ «Передача ТМЗ» делает движения только с видом операции «Передача сырья в переработку» по следующим регистрам:

  • Регистр бухгалтерии «Журнал проводок (бухгалтерский учет)». В качестве счета дебета выступает «Счет передачи (БУ)», в качестве счета кредита – «Счет учета (БУ)».
  • Регистр бухгалтерии «Журнал проводок (налоговый учет)». Если в документе установлен признак Учитывать КПН, то документ делает движения по налоговому учету. Проводки заполняются в соответствии с проводками бухгалтерского учета.

Для документа «Передача ТМЗ» предусмотрена возможность формирования печатной формы «Накладная на отпуск запасов (З-2)».
