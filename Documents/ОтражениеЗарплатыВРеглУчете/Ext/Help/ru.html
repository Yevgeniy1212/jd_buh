Документ «Отражение зарплаты в регл. учете»

Документ «Отражение зарплаты в регл. учете» предназначен для автоматизированного формирования проводок по бухгалтерскому и налоговому учету рассчитанной зарплаты.

Табличная часть «Отражение в учете» содержит проводки по отражению зарплаты в бухгалтерском учете.

Перед использованием документа, соответственно, необходимо:

  • Указать правила отражения отдельных видов начислений и удержаний в бухгалтерском учете в соответствующих планах видов расчета («Основные начисления организации», «Удержания организаций») – должна быть проставлена ссылка на нужный элемент справочника «Способы отражения зарплаты в регламентированном учете», где собственно и задается схема проводок для бухгалтерского учета;
  • Рассчитать зарплату за соответствующий период.

Для указания правил отражение расчетов по заработной плате в регламентированном учете предназначен справочник «Способы отражения зарплаты в регламентированном учете».

Предусмотрена возможность задания используемого способа отражения для разных уровней в зависимости от того, насколько детализируются затраты на оплату труда в составе себестоимости продукции:

  • Непосредственно в документах «Начисление зарплаты сотрудникам организации» и «Регистрация разовых удержаний сотрудников организаций» - можно задать способ отражения в бухгалтерском учете (реквизит «Способ отражения в бухучете» в табличной части документа), для этого нужно при нажатии на кнопку «Еще» на верхней панели документа, выбрать пункт «Изменить форму» и в появившемся окне установить признак Способ отражения в бухучете;
  • Если сотруднику ежемесячно начисляется определенный список видов начислений (плановый начисления, указываемые в документах «Прием на работу организацию» и «Кадровое перемещение организации»), которые относятся на разные счета затрат и/или разную аналитику затрат в бухгалтерском учете, то можно воспользоваться документом «Ввод сведений о регл. учете плановых начислений сотрудников организаций», позволяющий единожды задать применяемые способы отражения для комбинации значений: Сотрудник + Вид начисления;
  • Если в организации существуют виды начислений, которые отражаются в затратах одинаковым образом для всего персонала (вне зависимости от того сотрудника, который получает этот вид начисления), то проще всего указать способ отражения для такого вида непосредственно в форме его настройки (закладка «Учет» поле «Способ отражения в бухучете»). Для видов удержаний предусмотрен данный подход к указанию способа отражения в бухгалтерском учете;
  • В случае, если все затраты на оплату труда сотрудников одного подразделения организации относятся на один и тот же счет и аналитику затрат, то предусмотрена возможность задания способа отражения в целом для подразделения. Настройка порядка отражения начислений зарплаты для разных подразделений храниться в регистре сведений «Учет основного заработка сотрудников подразделений организации». Кроме того, установка способа отражения доступна непосредственно в форме редактирования справочника «Подразделения организаций»;
  • Если в процессе автоматического заполнения документа для каких-либо строк расчетов по заработной плате не будет найден ни один из вышеописанных способов отражения, то будет использован предопределенный элемент справочника «Способы отражения зарплаты в регламентированном учете» «Отражение начислений по умолчанию».

При автозаполнении документа «Отражение зарплаты в регл. учете» производится формирование бухгалтерских проводок по следующим операциям расчета заработной платы:

1. В соответствии с произведенной настройкой осуществляется поиск способа отражения для всех начислений заработной платы и удержаний из нее за расчетный период.

2. По данным об исчисленных за месяц обязательных пенсионных взносах из регистра накопления «ОПВ расчеты с фондами» формируются следующие проводки:

  • Дт 3350 - Кт 3220 - на сумму ОПВ;
  • Дт 7210 - Кт 3220 - на сумму пени за несвоевременное перечисление ОПВ.

3. По данным об исчисленном за месяц индивидуальном подоходном налоге из регистра накопления «ИПН расчеты с бюджетом» формируются проводки Дт 3350 - Кт 3120.

4. На основании информации из регистра накопления «СО расчеты с фондами» создаются проводки по социальным отчислениям:

  • Дт Счет затрат - Кт 3211 - на сумму исчисленных за месяц социальных отчислений. При этом счет затрат определяется по результатам отражения начислений заработной платы, с которых были рассчитаны социальные отчисления;
  • Дт 7210 - Кт 3211 - на сумму пени за несвоевременное перечисление СО.

5. По данным регистра накопления «СН исчисленный» формируются проводки начислению социального налога:

  • Дт Счет затрат - Кт 3150 - на сумму исчисленного социального налога. Порядок определения счета затрат, на который будет отнесен социальный налог, аналогичен порядку определения для социальных отчислений.

6. В случае если в организации создаются резервы по вознаграждения сотрудникам, то в процессе автоматического заполнения документа будут также сгенерированы проводки по формированию резервов. Подробнее порядок настройки расчета резервов описан в справочной информации к регистру сведений «Размеры отчислений в резервы».

7. По данным об исчисленных за месяц взносах на обязательное социальное медицинское страхование из регистра накопления «ВОСМС расчеты с фондами» формируются следующие проводки:

  • Дт 3350 - Кт 3212 - на сумму ВОСМС;
  • Дт 7210 - Кт 3212 - на сумму пени за несвоевременное перечисление ВОСМС.

8. На основании информации из регистра накопления «ООСМС расчеты с фондами» создаются проводки по отчислениям на обязательное социальное медицинское страхование:

  • Дт Счет затрат - Кт 3213 - на сумму исчисленных за месяц отчислений на обязательное социальное медицинское страхование. При этом счет затрат определяется по результатам отражения начислений заработной платы, с которых были рассчитаны отчисления на обязательное социальное медицинское страхование;
  • Дт 7210 - Кт 3213 - на сумму пени за несвоевременное перечисление ООСМС.
