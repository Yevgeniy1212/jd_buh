Документ «ОПВ Перечисление в фонды»

Документ предназначен для подготовки к перечислению удержанных из заработной платы сотрудников организации обязательных пенсионных взносов и обязательных профессиональных пенсионных взносов в накопительные пенсионные фонды. Вид перечисляемого взноса регулируется видами операций документа.

В шапке документа указываются реквизиты «Вид операции», «Структурная единица», «Вид платежа», «Подразделение» и «Фонд». Обязательно заполняется «Месяц исчисления».

Поле «Порядок заполнения» доступно для вида операции«Перечисление обязательных пенсионных взносов».

Для документа может быть выбран порядок заполнения:

  • «По сотрудникам» - табличная часть заполняется данными по работникам организации; 
  • «По прочим лицам» - табличная часть заполняется данными по прочим физическим лицам, в том числе по договорам ГПХ.

Документы могут быть заполнены следующими способами:

  • По задолженности за месяц - при выборе данного способа сумма к перечислению рассчитывается по задолженности за месяц, указанный в документе;
  • По остаткам задолженности на конец месяца - при выборе данного способа сумма к перечислению рассчитывается по остаткам задолженности на конец месяца заполнения.

Для перечисления обязательных пенсионных взносов возможен расчет сумм к перечислению по удельному весу. Если в настройках учетной политики налогового учета для организации снят признак При начислении ИПН и ОПВ принимать исчисленные суммы к учету как удержанные, то способы заполнения документа дополняются вариантами:

  • По задолженности за месяц, удержанной по НУ - при выборе данного способа сумма к перечислению рассчитывается по задолженности, рассчитанной по налоговому учету, за месяц, указанный в документе;
  • По остаткам задолженности на конец месяца, удержанной по НУ - при выборе данного способа сумма к перечислению рассчитывается по остаткам задолженности, рассчитанной по налоговому учету, на конец месяца заполнения.

Табличную часть документа можно заполнить автоматически (с помощью кнопки «Автозаполнение» командной панели формы документа). Если требуется рассчитать суммы, подлежащие перечислению, по предварительно введенному списку работников табличной части документа, то следует воспользоваться кнопкой «Рассчитать».

Для регистрации факта перечисления взносов на основании документа «ОПВ перечисление в фонды» необходимо ввести документ «Платежное поручение исходящее» или «Авансовый отчет». При возникновении ситуации возврата взносов, можно на основании документа «ОПВ Перечисление в фонды» создать документ «ОПВ Возврат взносов».
