Документ «Увольнение из организаций»

Документом регистрируется увольнение сотрудников из организации. Документом «прекращается» расчет для этих сотрудников всех назначенных им постоянных начислений и удержаний.

В табличной части документа заполняются следующие данные:

  • список увольняемых сотрудников
  • дата увольнения
  • статья ТК РК;
  • признак Прекращать стандартные вычеты.

 

ВАЖНО! В поле "Дата увольнения" указывается дата последнего рабочего дня. С даты, следующей за датой увольнения, сотруднику уже не будет начисляться заработная плата   (прекращаются его плановые начисления).

При проведении документа программа проверяет, не были ли сотрудники, указанные в документе, уволены ранее. При обнаружении таких ситуаций выдается сообщение об ошибке и документ не проводится.
