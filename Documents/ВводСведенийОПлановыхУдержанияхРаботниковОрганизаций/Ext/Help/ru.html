Документ «Ввод сведений о плановых удержаниях сотрудников организации»

Документ «Ввод сведений о плановых удержаниях сотрудников организации» предназначен для регистрации, изменении или прекращении плановых удержаний.

В табличной части «Сотрудники» указывается список сотрудников, по которым необходимо внести изменения. Заполнить табличную часть можно как списком сотрудников с различными отборами (кнопка «Заполнить»), так и подбором сотрудников из справочника «Сотрудники» (кнопка «Подбор»).

В табличной части «Удержания» указываются удержания сотрудников, а также действия, выполняемые с данными удержаниями:

  • Сотрудник – указывается сотрудник, для которого вводятся данные для учета плановых начислений;
  • Вид расчета - выбирается вид удержания из плана видов расчета «Удержания организаций»;
  • Действие - из списка выбирается действие назначения, изменения или прекращения удержания;
  • Дата начала, Дата окончания, Размер - указываются данные, необходимые для выполнения действий с удержанием;
  • Способ отражения в бухучете – указывается способ отражения указанных удержаний в бухгалтерском учете.

При выборе сотрудника табличная часть «Удержания» автоматически заполняется действующими на дату документа удержаниями подобранных сотрудников. Кнопка «По сотруднику» позволяется отобразить удержания по выбранному сотруднику.
