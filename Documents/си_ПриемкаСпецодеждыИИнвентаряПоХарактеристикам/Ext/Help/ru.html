Документ "Приемка спецодежды и инвентаря по характеристикам"

Конфигурация "1С-Рейтинг: Учет спецодежды и инвентаря".

Для регистрации информации о количестве поступившей спецодежды (инвентаря) на склад в разрезе характеристик используется документ «Приемка спецодежды и инвентаря по характеристикам» (раздел «Спецодежда и инвентарь» - «Эксплуатация спецодежды» - «Приемка спецодежды и инвентаря по характеристикам»).

В табличной части документа вводится количество поступившей спецодежды (инвентаря) с детализацией по характеристикам.

Важно! Если в настройках параметров учета спецодежды и инвентаря не стоит признак ведения складского учета в разрезе характеристик, то документ не доступен.

Если документ введен на основании документа «Поступление ТМЗ и услуг», то табличная часть документа автоматически заполняется перечнем поступившей спецодежды и инвентаря. В то же время, табличная часть документа может быть заполнена автоматически и без ввода документа на основании поступления. Для этого в документе предусмотрена кнопка «Заполнить», при нажатии на которую предлагается выбрать способ заполнения:

  • «Заполнить по поступлению» – табличная часть будет очищена и заполнена спецодеждой (инвентарем) документа поступления, выбранного из открывшегося журнала документов.
  • «Добавить из поступления»– спецодежда (инвентарь) из документа поступления будут добавлены в табличную часть, при этом табличная часть не очищается. Данный способ обычно используется для добавления в табличную часть номенклатурных позиций сразу из нескольких документов.
  • «Заполнить по основанию» – табличная часть будет очищена и заполнена спецодеждой (инвентарем) документа основания.

Для удобного заполнения характеристик спецодежды (инвентаря) в табличной части, предусмотрен инструмент группового изменения строк табличной части документа. Для вызова сервисного механизма используется кнопка «Изменить», располагающая на командной панели над табличной частью документа.

В  верхней табличной части обработки отображается поступившая спецодежда (инвентарь) и общее количество. В табличной части «Характеристики» указывается количество поступившей спецодежды, выбранной в табличной части «Товары», в разрезе характеристик.

Ввод на основании

Вводится на основании

Является основанием для

·       Поступление ТМЗ и услуг

·       Передача спецодежды и инвентаря в эксплуатацию

·       Перемещение спецодежды и инвентаря

Проведение документа

При проведении документа формируются записи в регистре по оперативному учету:

  • «Материалы на складе» – формируются движения по поступлению спецодежды и инвентаря на выбранный в документе склад (если используется складской учет) в разрезе характеристик (если ведется учет по характеристикам).

По бухгалтерскому учету  формируется запись

  •  Дт счета учета «Спецодежда и спецоснастка на складе»– Кт счета учета запасов (например, 1310).

Проводка по НУ будет сформирована только при установленном флаге «Учет КПН». Движения будут сформированы по счетам налогового учета (НУ) согласно установленным соответствиям счетам бухгалтерского учета (БУ).

Анализ данных

После проведения документа, с помощью отчета «Ведомость по спецодежде и инвентарю на складе» можно проанализировать остатки спецодежды и инвентаря в различных разрезах (по характеристикам, складам, и т.д.).

Проанализировать данные по бухгалтерскому учету  можно с помощью типовых отчетов «Оборотно-сальдовая ведомость по счету», «Карточка счета».
