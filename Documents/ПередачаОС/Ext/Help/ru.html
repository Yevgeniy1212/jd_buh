Документ «Передача ОС»

Документ предназначен для оформления продажи основных средств. Документ отражается по бухгалтерскому, налоговому учету. При проведении документа выполняется несколько действий:

  • доначисление амортизации за месяц выбытия;
  • списание основного средства с учета;
  • отражение задолженности по взаиморасчетам.

Вид операции, которая будет произведена в результате проведения документа, определяется непосредственно в самой форме документа в поле «Вид операции». При выборе вида операции автоматически формируется и соответствующее название документа и при необходимости изменяется состав граф табличной части документа. В любой момент времени можно изменить вид операции (даже для проведенного документа) и конфигурация корректно пересчитает все данные в информационной базе.

Для вида операции доступны следующие значения:

  • реализация;
  • передача структурному подразделению.

Продажа ОС осуществляется документов, в котором установлен вид операции «Реализация».

Передача ОС структурному подразделению осуществляется документом, в котором установлен вид операции «Передача структурному подразделению».

Реквизиты шапки документа

В шапке документа указываются следующие реквизиты:

  • Структурная единица – структурная единица, по учету которой проводится операция;
  • Контрагент - покупатель основного средства;
  • Договор - договор с контрагентом.

Табличная часть «ОС»

В табличной части устанавливается перечень продаваемых основных средств. Для каждого основного средства устанавливается стоимость реализации, сумма НДС, ставка НДС. Стоимость продажи указывается в валюте взаиморасчетов договора с контрагентом. Если договор выбран в валюте, отличной от валюты регламентированного учета в форме «Цены и валюта» становятся доступны поля по изменению валюты документа и курса взаиморасчетов. 

Для каждого вида учета заполняются по кнопке «Заполнить» стоимостные данные по основному средству:

  • Стоимость (БУ) - стоимость основного средства на момент продажи;
  • Остаточная стоимость (БУ) - разница между стоимостью и начисленной амортизацией;
  • Амортизация (БУ) - амортизация, начисленная за весь период эксплуатации основного средства;
  • Амортизация за месяц (БУ) - сумма амортизации, которая должна быть начислена за месяц, в котором отражается операция продажи;
  • Сумма – указывается сумма реализации ОС;
  • Сумма НДС - рассчитывается сумма НДС;
  • Всего – отражается итоговая сумма реализации ОС.

Стоимостные показатели (в том числе и амортизация за месяц) могут быть рассчитаны автоматически при нажатии на кнопку «Заполнить».

Для быстрого заполнения табличной части однотипными объектами основных средств, имеющими одинаковые наименования, нужно ввести в табличную часть хотя бы один такой объект. Затем следует воспользоваться кнопкой «Заполнить» в командной панели табличной части. При выборе в выпадающем меню пункта «По наименованию» табличная часть будет заполнена объектами основных средств, имеющими такое же наименование, как у введенного первоначально.

При выбытии, объект снимается с учета по налогам - на имущество, транспорт, земельному налогу, для него устанавливается признак выбытия в амортизационной группе НУ (если ранее основное средство являлось объектом налогообложения). С даты передачи начисление имущественного, транспортного и земельного налога будет прекращено.
