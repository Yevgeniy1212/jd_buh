Документ «Ввод сведений о регл. учете плановых начислений сотрудников организации».

Документ «Ввод сведений о регл. учете плановых начислений сотрудников организации» предназначен для регистрации изменений в отражении постоянных начислений сотрудников организации в регламентированном учете.

  • «Сотрудник» - сотрудник, для которого вводятся данные для отражения начислений в регламентированном учете;
  • «Вид расчета» - выбирается вид начисления из плана видов расчета «Начисления организации»;
  • «Дата начала» - дата, начиная с которой данные строки будут действовать.
  • «Способ отражения в бухучете» - способ отражения указанных начислений в регламентированном учете.

Документ может быть заполнен на основании следующих документов:

  • Документ «Кадровое перемещение организации»;
  • Документ «Отражение зарплаты в регл учете»;
  • Документ «Прием на работу в организацию».
