Документ «Перемещение товаров»

Документ «Перемещение ТМЗ» предназначен для оформления перемещения товарно-материальных запасов (товаров, материалов, готовой продукции и прочего) между складами.

В качестве данных отправителя указывается «Склад», с которого происходит списание товаров, в качестве данных получателя указывается «Склад», на который поступают товары. Если организация ведет учет по структурным подразделениям, не выделенным на отдельный баланс, товары можно перемещать между структурными подразделениями одной организации с одного склада на другой. При перемещении товаров структурному подразделению, не выделенному на отдельный баланс, необходимо указывать значения реквизитов «Структурная единица» отправителя или получателя.

При проведении документа одновременно выполняется списание товара со склада и структурного подразделения отправителя и поступление на склад и структурное подразделение получателя.

При создании нового документа поля «Организация» и «Склад» заполняются автоматически значениями, указанными в настройках пользователя в группе «Основные значения для подстановки в документы и справочники». Данные поля являются обязательными для заполнения.

Признак «Учитывать КПН» влияет на отражение данных документа по налоговому учету. Если в учетной политике по налоговому учету для указанной организации снят признак «Плательщик налога на прибыль», то признак «Учитывать КПН» и все счета налогового учета будут не доступны.

Документ может быть оформлен как свободный документ, так и на основании других документов:

  • На основании документа «Отчет производства за смену»;
  • На основании документа «Поступление ТМЗ и услуг».

Если документ «Перемещение ТМЗ» оформляется на основании перечисленных документов, его табличная часть автоматически заполняется данными из документа-основания.

Кнопка «Заполнить» предназначена для заполнения табличной части документа остатками номенклатурных позиций на указанном в шапке документа складе.
Если документ оформляется как свободный документ, его табличная часть заполняется построчным вводом (при помощи кнопки «Добавить») или подбором (при помощи кнопки «Подбор») товаров из справочника «Номенклатура». При подборе можно установить вариант подбора, при котором будут показываться только те товары, которые имеются в наличии на складе «Отправитель» (вариант подбора «По остаткам номенклатуры»).

Кнопка «Изменить» предназначена для вызова обработки «Обработка табличной части Товары» для группового изменения табличной части «ТМЗ».

При выборе номенклатуры поля «Счет отпр. (БУ)», «Счет отпр. (НУ)», «Счет получ. (БУ)» и «Счет получ. (НУ)» устанавливаются автоматически из регистра сведений «Счета учета номенклатуры».

Табличная часть "Источники происхождения" - используется для перемещения товара по источникам происхождения в регистре накопления "Товары организаций". Заполняется, если в конфигурации используется учет товаров по источникам происхождения:

  • Номенклатура - заполняется из табличной части "ТМЗ"
  • Источник происхождения - исходный источник происхождения товара
  • Источник происхождения получателя - новый источник происхождения товара

Если не меняются структурная единица и источники происхождения, то движения по регистру "Товары организаций (БУ)" не выполняются.

Закладка «Дополнительно» предназначена для ввода дополнительной информации по документу:

  • Ответственный. Из справочника «Пользователи» указывается ответственный за документ пользователь. При создании нового документа заполняется автоматически данными текущего пользователя.
  • Документ-основание. Указывается документ, на основании которого был создан или заполнен документ «Перемещение ТМЗ». При изменении значения документа-основания предоставляется возможность перезаполнения данных документа данными документа-основания.

Документ «Перемещение ТМЗ» формирует движения по следующим регистрам:

  • Регистр бухгалтерии «Журнал проводок (бухгалтерский учет)». В качестве счета дебета выступает «Счет получ. (БУ)», в качестве счета кредита – «Счет отпр. (БУ)».
  • Регистр бухгалтерии «Журнал проводок (налоговый учет)». Если в документе установлен признак «Учитывать КПН», то документ делает движения по налоговому учету. Проводки заполняются в соответствии с проводками бухгалтерского учета.

На основании документа «Перемещение ТМЗ» могут быть оформлены следующие документы:

  • Документ «Списание ТМЗ»;
  • Документ «Требование – накладная»;
  • Документ «Сторнирование».

Для документа «Перемещение ТМЗ» предусмотрена возможность формирования печатных форм «Накладная на перемещение» и «Накладная на внутреннее перемещение запасов (З-5)».
