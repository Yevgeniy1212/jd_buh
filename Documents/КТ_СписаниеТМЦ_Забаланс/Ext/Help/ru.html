Документ «Списание ТМЗ»

Документ «Списание ТМЗ» предназначен для оформления списания товаров.

При создании нового документа поля «Структурная единица» и «Склад» заполняются автоматически значениями, указанными в настройках пользователя в группе «Основные значения для подстановки в документы и справочники». Данные поля являются обязательными для заполнения.

Признак Учитывать КПН влияет на отражение данных документа по налоговому учету. Если в учетной политике по налоговому учету для указанной организации снят признак Плательщик налога на прибыль, то признак Учитывать КПН и все счета налогового учета будут недоступны.

Документ может быть оформлен как свободный документ (в случае произвольного списания товаров), так и на основании других документов:

  • документа «Инвентаризация ТМЗ на складе»;
  • документа «Перемещение ТМЗ»;
  • документа «Поступление ТМЗ и услуг»;
  • документа «Авансовый отчет».

Списание товаров производится по себестоимости, рассчитанной на момент оформления документа «Списание ТМЗ».

Закладка «ТМЗ». Если документ «Списание ТМЗ» оформляется на основании перечисленных документов, его табличная часть автоматически заполняется данными из документа-основания.

Если документ оформляется как свободный документ, его табличная часть заполняется построчным вводом (при помощи кнопки «Добавить») или подбором (при помощи кнопки «Подбор») товаров из справочника «Номенклатура». При подборе товаров из справочника можно установить вариант подбора «По остаткам номенклатуры». Тогда в списке будут отображены только те товары, которые имеются в наличии на указанном в шапке документа складе.

Табличная часть "Источники происхождения" служит для указания источников происхождения у товаров. Заполняется, если в конфигурации используется учет товаров по источникам происхождения.

При выборе номенклатуры поля «Счет учета (БУ)», «Счет списания затрат (БУ)», «Счет учета (НУ)», «Счет списания затрат (НУ)» и соответствующая аналитика заполняются автоматически из регистра сведений «Счета учета номенклатуры».

Закладка «НДС». В случае необходимости, при списании ТМЗ может быть скорректирована сумма НДС, ранее отнесенная в зачет. Для проведения данной операции предназначен признак Корректировка НДС, ранее принятого к зачету. В этом случае на закладке «НДС» необходимо указать счет отнесения затрат по НДС и его аналитику, а так же аналогичные реквизиты для отражения затрат в налоговом учете (для документов, отражаемых в учете по налогу на прибыль).

Если признак Корректировка НДС, ранее принятого к зачету установлен, то в табличной части «ТМЗ» появляются дополнительные поля: «Ставка НДС», «Вид оборота» и «Вид поступления».

Если в учетной политике по налоговому учету для организации снят признак Плательщик налога на добавленную стоимость, то соответственно операция корректировки НДС будет не доступна.

Закладка «Дополнительно». Закладка «Дополнительно» предназначена для ввода дополнительной информации по документу:

  • «Ответственный». Из справочника «Пользователи» указывается ответственный за документ пользователь. При создании нового документа заполняется автоматически данными текущего пользователя.
  • «Документ-основание». Указывается документ, на основании которого был создан или заполнен документ «Списание ТМЗ». При изменении значения документа-основания предоставляется возможность перезаполнения данных документа данными документа-основания.
  • «Основание списание». Заполняется вручную по необходимости причиной списания товаров.
  • Табличная часть «Комиссия». Указываются члены комиссии и председатель из справочника «Физические лица».

Документ «Списание ТМЗ» формирует движения по следующим регистрам:

Регистр бухгалтерии «Журнал проводок (бухгалтерский учет)». В качестве счета дебета выступает «Счет списания затрат (БУ)», в качестве счета кредита – «Счет учета (БУ)».
Регистр бухгалтерии «Журнал проводок (налоговый учет)». Если в документе установлен признак Учитывать КПН, то документ делает движения по налоговому учету. Проводки заполняются в соответствии с проводками бухгалтерского учета.
Регистр накопления «Корректировка НДС к возмещению». Если на закладке «НДС» установлен признак Корректировка НДС, ранее принятого к зачету, документ делает движения по регистру.

В поле «Комментарий» можно ввести любую дополнительную информацию. Весь текст комментария можно просмотреть или отредактировать, нажав на кнопку «Выбрать».

На основании документа «Списание ТМЗ» может быть оформлен документ «Регистрация прочих операция по приобретенным товарам (работам, услугам) в целях НДС».

Для документа «Списание ТМЗ» предусмотрена возможность формирования печатных форм:

  • Списание товаров;
  • Акт списания товаров;
  • Акт списания запасов (З-6).
