Документ «Партия (ручной учет)»

Вспомогательный документ, который не делает никаких проводок, предназначен для ведения ручного учета по партиям, т.е. для организации субконто «Партия» на счетах учета ТМЗ.

Реквизиты необходимые для заполнения:

  • Номер;
  • Дата;
  • Структурная единица;
  • Контрагент;
  • Договор;
  • Комментарий.
