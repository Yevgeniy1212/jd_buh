Документ «Ввод начальных остатков по зарплате»

Документ «Ввод начальных остатков по зарплате» предназначен для ввода сведений по взаиморасчетам, связанным с заработной платой, необходимых для дельнейшего ведения учета. Документ рекомендуется вносить последним днем месяца, за который рассчитаны данные. Для упрощения ввода данных желательно, чтобы перед началом занесения остатков уже были оформлены приемы на работу сотрудников организации.

На закладке «Зарплата и налоги» указывается список сотрудников организации, по которым есть задолженность по заработной плате, по депонированным суммам или по задолженности по индивидуальному подоходному налогу в налоговом учете.

  • «Сотрудник» - сотрудник, для которого вводятся данные для учета;
  • «Месяц» - определяется месяц предыдущего периода (до начала ведения учета в конфигурации), к которому относятся данные строки;
  • «Задолженность по зарплате» - сумма задолженности перед сотрудником (вносится как положительное значение) или задолженность сотрудника перед организацией (вносится как отрицательное значение). После проведения документа данная сумма будет отражена как кредитовый остаток счета 3350 «Краткосрочная задолженность по оплате труда»;
  • «Задолженность по депонированным суммам» - сумма задолженности перед сотрудником (вносится как положительное значение) или задолженность сотрудника перед организацией (вносится как отрицательное значение). После проведения документа данная сумма будет отражена как кредитовый остаток счета 3384 «Задолженность по депонированной заработной плате»;
  • «Задолженность по ИПН (НУ)» - сумма индивидуального подоходного налога, исчисленная в соответствующем месяце, но не отражена в налоговой отчетности, в связи с тем, что соответствующие доходы еще не были выплачены сотруднику. Следует обратить внимание, что данная сумма не отражается в бухгалтерском учете, поэтому остатки по счету 3120 «Индивидуальный подоходный налог» следует ввести с помощью документа «Операция (бухгалтерский и налоговый учет)».

На закладке «Взносы и отчисления» указывается список сотрудников организации, по которым есть задолженность по обязательным пенсионным взносам, социальным отчислениям, обязательным профессиональным пенсионным взносам, взносам и отчислениям на обязательное социальное медицинское страхование

  • «Сотрудник» - сотрудник, для которого вводятся данные для учета;
  • «Месяц» - определяется месяц предыдущего периода (до начала ведения учета в конфигурации), к которому относятся данные строки;
  • «Вид платежа» - вид платежа, к которому относятся данные строки;
  • «Задолженность по ОПВ» - сумма задолженности по не перечисленным (вносится как положительное значение) или излишне перечисленным (вносится как отрицательное значение) обязательным пенсионным взносам сотрудников. После проведения документа данная сумма будет отражена как кредитовый остаток счета 3220 «Обязательства по пенсионным отчислениям»;
  • «Задолженность по СО» - сумма задолженности по не перечисленным (вносится как положительное значение) или излишне перечисленным (вносится как отрицательное значение) социальным отчислениям на сотрудника. После проведения документа данная сумма будет отражена как кредитовый остаток счета 3211 «Обязательства по социальному страхованию»;
  • «Задолженность по ОППВ» - сумма задолженности по не перечисленным (вносится как положительное значение) или излишне перечисленным (вносится как отрицательное значение) обязательным профессиональным пенсионным взносам сотрудников. После проведения документа данная сумма будет отражена как кредитовый остаток счета 3220 «Обязательства по пенсионным отчислениям»
  • «Задолженность по ОПВР» - сумма задолженности по не перечисленным (вносится как положительное значение) или излишне перечисленным (вносится как отрицательное значение) обязательным пенсионным взносам работодателя. После проведения документа данная сумма будет отражена как кредитовый остаток счета 3250 «Обязательства по обязательным пенсионным взносам работодателя»
  • «Задолженность по взносам ОСМС» - сумма задолженности по не перечисленным (вносится как положительное значение) или излишне перечисленным (вносится как отрицательное значение) взносам на обязательное социальное медицинское страхование сотрудников. После проведения документа данная сумма будет отражена как кредитовый остаток счета 3212 «Обязательства по взносам на социальное медицинское страхование»
  • «Задолженность по отчислениям ОСМС» - сумма задолженности по не перечисленным (вносится как положительное значение) или излишне перечисленным (вносится как отрицательное значение) отчислениям на обязательное социальное медицинское страхование сотрудников. После проведения документа данная сумма будет отражена как кредитовый остаток счета 3213 «Обязательства по отчислениям на социальное медицинское страхование»
