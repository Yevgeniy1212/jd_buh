Документ «СО Перечисление в фонды»

Документ предназначен для подготовки к перечислению исчисленных обязательных социальных отчислений в ГФСС.

В шапке документа указываются реквизиты «Структурная единица», «Подразделение организации», «Вид операции» , « Порядок заполнения» , «Вид платежа», «Фонд».

Поле «Порядок заполнения» доступно для видов операции: «Перечисление обязательных социальных отчислений» и «Перечисление взносов ОСМС».

Для документа может быть выбран порядок заполнения:

  • «По сотрудникам» - табличная часть заполняется данными по работникам организации; 
  • «По прочим лицам» - табличная часть заполняется данными по прочим физическим лицам, в том числе по договорам ГПХ.

Документ может быть заполнен следующими способами:

  • «По задолженности за месяц» - при выборе данного способа сумма к перечислению рассчитывается по задолженности за месяц, указанный в документе;
  • «По остаткам задолженности на конец месяца» - при выборе данного способа сумма к перечислению рассчитывается по остаткам задолженности на конец месяца заполнения.

Табличную часть документа можно заполнить автоматически (с помощью кнопки «Автозаполнение» командной панели формы документа). Если требуется рассчитать суммы, подлежащие перечислению, по предварительно введенному списку работников табличной части документа, то следует воспользоваться кнопкой «Рассчитать».

В правой части формы документа расположена контекстно-зависимая справка, в которой приводятся дополнительные разъяснения параметров заполнения документа.

Для регистрации факта перечисления взносов на основании документа «СО перечисление в фонды» необходимо ввести документ «Платежное поручение (исходящее)» или «Авансовый отчет».

В случае возврата ранее перечисленных социальных отчислений, на основании документа «СО перечисление в фонды» можно создать документ «СО возврат отчислений».
