Отражение зарплаты в бух. учете (интеграция с ЗУП 3)

Для хранения способов отражения заработной платы в регламентированном учете в виде шаблонов корреспонденции счетов предназначен справочник Способы отражения зарплаты в бух. учете (интеграция с ЗУП 3).

Документ Отражение зарплаты в бух. учете (интеграция с ЗУП 3) предназначен для автоматизированного формирования проводок по бухгалтерскому учету рассчитанной зарплаты. Для документа не предусмотрено автоматическое заполнение данными. Такой подход обусловлен получением данных из конфигурации ЗУП 3 при выполнении синхронизации. Проводки по отражению заработной платы в бухгалтерском учете формируются в момент проведения документа, в табличных частях сведения по проводкам бухгалтерского и налогового учетов отсутствуют.

Виды операций в табличных частях документа определяют порядок отражения в регламентированном учете в зависимости от вида начисления, удержания, налога, взноса и отчисления.

В качестве счета учета используется счет, указанный в способе отражения зарплаты в бухгалтерском учете, либо счет по умолчанию (предопределенный счет 7210 "Административные расходы").

В качестве счета учета по налогам используется счет, определенный в справочнике Налоги, сборы, отчисления по виду налога, либо по умолчанию соответствующий предопределенный счет.
