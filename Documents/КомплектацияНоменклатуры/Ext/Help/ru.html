Документ «Комплектация ТМЗ»

Документ «Комплектация ТМЗ» предназначен для отражения операций по комплектации или разукомплектации запасов, состоящих из отдельных комплектующих.

Документом можно оформить следующие виды операций:

  • Комплектация;
  • Разукомплектация.

Вид операции «Комплектация» предназначен для формирования комплекта из списка комплектующих. При проведении документа происходит списание комплектующих и поступление готового комплекта на склад. При этом списание комплектующих производится по их себестоимости, а поступление комплекта на склад производится по себестоимости, равной сумме себестоимости комплектующих.

Вид операции «Разукомплектация» предназначен для списания комплекта со склада и поступления комплектующих на склад. Себестоимость комплектующих, по которой они поступают на склад, рассчитывается в соответствии с себестоимостью комплекта в той доле, которая указана для комплектующего.

Вид операции определяется в шапке документа в поле «Вид операции». Поле выделено специальным цветом. При выборе вида операции автоматически меняется внешний вид документа (изменяется состав элементов шапки документа и граф табличных частей). Вид операции желательно указывать при вводе нового документа. Но в любой момент времени его можно изменить (даже для проведенного документа) и конфигурация автоматически пересчитает все данные в информационной базе.

При создании нового документа поля «Структурная единица» и «Склад» заполняются автоматически значениями, указанными в настройках пользователя в группе «Основные значения для подстановки в документы и справочники».

Признак Учитывать КПН влияет на отражение данных документа по налоговому учету. Если в учетной политике по налоговому учету для указанной организации снят признак Плательщик налога на прибыль, то признак Учитывать КПН и все счета налогового учета будут не доступны.

В шапке документа из справочника «Номенклатура» выбирается комплект, который необходимо собрать или разукомплектовать. Счета учета бухгалтерского и налогового учета заполняются автоматически из регистра сведений «Счета учета номенклатуры» после выбора номенклатурной позиции. При необходимости можно изменить корреспонденцию счетов, указав другие счета для комплекта. Указывается количество комплектов и единица измерения.

На закладке «Комплектующие» заполняется список комплектующих. Поле «Доля стоимости» доступно только с видом операции «Разукомплектация», которое предназначено для определения, в каких долях стоимость разбираемого комплекта будет разделена между входящими в него комплектующими. Счета учета устанавливаются автоматически из регистра сведений «Счета учета номенклатуры». При необходимости можно изменить корреспонденцию счетов, указав другие счета для комплектующих. Закладка может быть заполнена построчным вводом (при помощи кнопки «Добавить») или подбором (при помощи кнопки «Подбор») товаров из справочника «Номенклатура». Список комплектующих можно заполнить автоматически из справочника «Спецификация номенклатуры» (кнопка «Заполнить» - «Заполнить по спецификации»).

Пример. Себестоимость комплекта равна 1000 тенге. Комплект состоит из трех деталей. Для первой детали установили долю - 1, для второй - 5, для третьей - 4. На склад первая деталь поступит по себестоимости 100 тенге, вторая – 500 тенге, третья – 400 тенге.

Реквизит "Источник происхождения" комплекта заполняется, если в конфигурации используется учет товаров по источникам происхождения.

Табличная часть "Источники происхождения" служит для указания источников происхождения у комплектующих. Заполняется, если в конфигурации используется учет товаров по источникам происхождения.


При проведении документ «Комплектация ТМЗ» делает движения по следующим регистрам:

  • Регистр бухгалтерии «Журнал проводок (бухгалтерский учет)». При комплектации номенклатуры для каждой строки табличной части «Комплектующие» формируется отдельная проводка по дебету счета учета комплекта и кредиту счета учета комплектующей. При разукомплектации номенклатуры для каждой строки табличной части «Комплектующие» формируется отдельная проводка по дебету счета учета комплектующей и кредиту счета учета комплекта.
  • Регистр бухгалтерии «Журнал проводок (налоговый учет)». Если в документе установлен признак Учитывать КПН, то документ делает движения по налоговому учету.

Для документа «Комплектация ТМЗ» предусмотрена печатная форма «Комплектация номенклатуры».
