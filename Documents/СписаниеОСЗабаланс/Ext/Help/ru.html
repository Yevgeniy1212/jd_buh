Документ «Списание ОС»

Документ «Списание ОС» предназначен для отражения в учете выбытия основного средства в связи с моральным и физическим износом, а также при ликвидации в связи с чрезвычайной ситуацией.

Выбывающие объекты выбираются из справочника «Основные средства».
Для более быстрого заполнения табличной части документа можно воспользоваться кнопкой «Подбор».

После формирования списка основных средств, можно воспользоваться кнопкой «Заполнить» для заполнения табличной части информацией об объектах основных средств по данным учета. Сумма амортизации за текущий месяц определяется расчетным путем в документе.

Для быстрого заполнения табличной части объектами основных средств, имеющими одинаковые наименования, необходимо ввести в табличную часть один объект и затем воспользоваться кнопкой «Заполнить» на командной панели табличной части. При выборе в выпадающем меню пункта «По наименованию», табличная часть будет заполнена объектами основных средств, имеющими такое же наименование, как и у введенного первоначально.

При списании с учета указанных в табличной части основных средств формируются проводки:

  • по доначислению амортизации, если это необходимо;
  • по списанию амортизационных начислений;
  • по списанию остаточной стоимости объекта на указанный в форме счет учета расходов.

При необходимости, основное средство можно снять с учета по налогам и установить признак выбытия в амортизационной группе НУ (если ранее основное средство являлось объектом налогообложения). Для этого в табличной части следует установить признак Снять с учета по налогам.

На закладке «Счет списания затрат/отнесения доходов» указываются данные по счетам затрат /доходов (БУ и НУ), статьям затрат/доходов, подразделениям и т.д для корректного отражения результата списания и отнесения доходов, либо затрат на необходимые счета.

Закладка «ТМЗ» заполняется в случае, если при списании основного средства у нас образуются ТМЗ, которые необходимо оприходовать и отразить в учете.

На закладке отображаются сведения:

  • «Основное средство» – основное средство, подлежащее списанию;
  • «Склад» – склад, на который будут оприходованы ТМЗ;
  • «Номенклатура» – приходуемые ТМЗ;
  • «Количество» – количество приходуемых ТМЗ;
  • «Цена» – стоимость оприходования ТМЗ;
  • «Сумма» – произведение количества и цены;
  • «Счет БУ/НУ» – счета, на которые будут оприходованы ТМЗ.

Закладка «Дополнительно» содержит данные, необходимые в том числе для заполнения печатной формы ДА-3 «Акт на выбытие (списание) долгосрочных активов».

Формирование печатной формы ДА-3 осуществляется с помощью кнопки «Печать» в нижней части окна документа.
