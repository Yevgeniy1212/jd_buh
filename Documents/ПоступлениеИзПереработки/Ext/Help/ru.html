Документ «Поступление из переработки»

Документ предназначен для отражения в бухгалтерском и налоговом учете операции получения услуг по выпуску продукции сторонним переработчиком, самой продукции и отражения отчета об использованных материалах.

Общие реквизиты документа

  • Склад - место хранения, на который поступает готовая продукция. Значение реквизита выбирается из справочника «Склады». Реквизит является обязательным для заполнения;
  • Контрагент - переработчик сырья. Значение реквизита выбирается из справочника «Контрагенты». Реквизит является обязательным для заполнения;
  • Договор - договор переработчика с видом «С поставщиком»;
  • Подразделение на котором будет отражено производство продукции силами стороннего переработчика;
  • Номенклатурная группа, к которой отнесена продукция, произведенная переработчиком;
  • Счет затрат - счета основного или вспомогательного производства, на котором будет отражено производство продукции силами стороннего переработчика.

На закладке «Продукция» указывается перечень продукции, произведенной переработчиком.

  • Номенклатура - продукция, выпущенная сторонним переработчиком. Значение реквизита выбирается из справочника «Номенклатура» и является обязательным для заполнения;
  • Спецификация - спецификация продукции, если для данной продукции заведена спецификация. Значение реквизита выбирается из справочника «Спецификации номенклатуры». Такая информация позволит автоматически рассчитать состав материалов, необходимый для производства указанной продукции заполнить закладку «Материалы»;
  • Количество - количество продукции;
  • Цена (плановая) - плановая цена производства продукции сторонним переработчиком;
  • Сумма (плановая) - плановая стоимость услуги по выпуску продукции сторонним переработчиком;
  • Счет учета (БУ) - счет учета готовой продукции в бухгалтерском учете;
  • Счет учета (НУ) - счет учета готовой продукции в налоговом учете.

На закладке «Услуги» указывается перечень услуг, оказанных переработчиком по переработке сырья в готовую продукцию.

  • Номенклатура - оказанная услуга. Значение реквизита выбирается из справочника «Номенклатура» и является обязательным для заполнения;
  • Содержание услуги, доп. сведения - дополнительная информация об оказанной услуге;
  • Статья затрат - статья затрат, по которой услуги по переработке будут отнесены на затраты производства. Значение реквизита выбирается из справочника «Статьи затрат»;
  • Количество - количество оказанных услуг;
  • Цена - цена оказанной услуги;
  • Сумма - стоимость оказанной услуги;
  • Ставка НДС;
  • Сумма НДС;
  • Счет НДС - счет учета для отражения суммы НДС по приобретению для бухгалтерских проводок;
  • Вид оборота - аналитика для регистра «НДС к возмещению»;
  • Вид поступления для - аналитика для регистра «НДС к возмещению».

На закладке «Материалы» указываются материалы, которые были использованы переработчиком при выпуске продукции.

  • Номенклатура - материал, переданный в переработку. Значение реквизита выбирается из справочника «Номенклатура» и является обязательным для заполнения;
  • Статья затрат - статья затрат, на которую был списан материал;
  • Количество - количество материала;
  • Счет учета (БУ) - счет бухгалтерского учета отданных в переработку материалов (например, 1351). Значение реквизита выбирается из плана счетов бухгалтерского учета и заполняется автоматически при выборе продукции на основании данных регистра сведений «Счета учета номенклатуры организаций»;
  • Счет учета (НУ) - счет налогового учета отданных в переработку материалов.

Предусмотрено заполнение данной табличной части по кнопке «Заполнить». Заполнение производится материалами, которые согласно спецификации необходимы для производства продукции, указанной на закладке «Продукция» или по остаткам счета «Материалы переданные в переработку».

На закладке «Возвращенные материалы» указываются материалы, которые были возвращены переработчиком, потому что не были использованы при выпуске продукции.

  • Номенклатура - материал, возвращаемый из переработки. Значение реквизита выбирается из справочника «Номенклатура» и является обязательным для заполнения;
  • Количество - количество материала;
  • Счет учета (БУ) - счет бухгалтерского учета отданных в переработку материалов (например, 1351). Значение реквизита выбирается из плана счетов бухгалтерского учета и заполняется автоматически при выборе продукции на основании данных регистра сведений «Счета учета номенклатуры организаций»;
  • Счет передачи (БУ) - счет бухгалтерского учета для передачи материалов. Значение реквизита выбирается из плана счетов бухгалтерского учета и заполняется автоматически при выборе продукции на основании данных регистра сведений «Счета учета номенклатуры».
  • Источник происхождения - источник происхождения материала. Значение реквизита выбирается из справочника "Источники происхождения". Заполняется, если в конфигурации используется учет товаров по источникам происхождения.

Предусмотрено заполнение данной табличной части по кнопке «Заполнить». Заполнение производится материалами, которые согласно спецификации необходимы для производства продукции, указанной на закладке «Продукция» или по остаткам счета «Материалы переданные в переработку».

На закладке «Счета учета расчетов» в документе заполняются «Счет учета расчетов с контрагентом» и «Счет учета расчетов по авансам». Эти счета заполняются в соответствии с заданными в регистре сведений значениями по умолчанию для каждого контрагента. При необходимости пользователь может изменить корреспонденцию счетов.

Особенности проведения документа

По данным табличной части «Продукция» производится поступление из производства на склад готовой продукции в плановых ценах.

По данным табличной части «Услуги» производится принятие услуг переработки сырья сторонней организацией на расходы основного или вспомогательного производства.

По данным табличной части «Материалы» производится списание на затраты основного или вспомогательного производства материалов, ранее переданных в переработку на сторону и учтенных на счете 1351 «Материалы, переданные в переработку».

Принятие НДС к зачету

При "стандартной" схеме, т.е. когда налоговый счет-фактура получен вместе с первичными документами.
При проведении документа будут сформированы бухгалтерские проводки по дебету счета 1421 "Налог на добавленную стоимость" и движения по регистру накопления "НДС к возмещению".

Отражение "нестандартной" схемы отнесения НДС в зачет.

Ситуация по отложенному принятию НДС к зачету возникает в случае, когда налоговый счет-фактура получен позднее первичных документов. В этом случае устанавливается признак "Отложить принятие НДС к зачету" в форме "Цена и валюта" по ссылке "Тип цен".
Будет сформирована бухгалтерская проводка по дебету счета 1422 "Налог на добавленную стоимость (отложенное принятие к зачету)". Движения по регистру накопления "НДС к возмещению" будут отсутствовать.
