Справочник "Организации"

Справочник вызывается для ввода и редактирования информации из пункта «Главное меню» / «Все функции» / «Справочники» / «Организации».

Справочник «Организации» предназначен для ведения списка собственных организаций, входящих в состав предприятия (группы), и хранения постоянных сведений о них.

Реквизиты:
  • Наименование - задается пользователем самостоятельно в зависимости от используемого в организации подхода.
  • Код – заполняется автоматически при записи элемента справочника, а также может быть откорректирован вручную.
  • Префикс – устанавливается префикс - три буквенных символа в начала каждого номера документов данной организации.
  • Полное наименование – для размещения полного официального наименования организации.


Отображается только в том случае, если контрагент является Юридическим лицом:

  • Отделение иностранной организации (зависимый агент) – устанавливается, если организация является иностранной организацией, в этом случае появляется закладка «Иностранная организация», в которой нужно заполнить сведения об иностранной организации.


Отображается только в том случае, если контрагент является Физическим лицом:

  • Физическое лицо - дополнительный реквизит, выбирается из справочника Физические лица.

Закладка Основные:
  • БИН / ИИН - бизнес - идентификационный номер (для юридического лица) / индивидуальный идентификационный номер (для физического лица).
  • КБЕ – Код бенефициара и сектора экономики.
  • Дата регистрации - дата регистрации в органах юстиции.
  • Налоговый комитет – Налоговый комитет по месту регистрации предприятия.
  • Относить налоги к бюджету городов районного значения, сел, поселков, сельских округов - если данный признак установлен, то для видимости будет доступен выбор Аппарата акимата для форм налоговой отчетности.
  • Свидетельство о постановке на учет по НДС - серия, номер, дата выдачи и дата постановки на учет по НДС.
  • Указывать реквизиты головной организации в счете фактуре - позволяет организовать автоматическую подстановку значения головной организации в качестве поставщика/покупателя в счете фактуре.

Контроль дубликатов в справочнике осуществляется по полям «ИИН/БИН» автоматически при вводе данных реквизитов. Если ИИН/БИН введены некорректно, выходит соответствующее предупреждение.

Закладка Банковские счета:

В таблице «Банковские счета» отображаются все банковские счета организации.

Список банковских счетов может дополняться.

Жирным шрифтом отмечается основной банковский счет, который можно изменить соответствующей кнопкой командной панели «Установить основным», в этом случае он будет подставляться в документы по умолчанию.

Закладка Кассы:

В таблице «Кассы» отображаются все кассы организации.

Список касс может дополняться.

Жирным шрифтом отмечается основная касса, которую можно изменить соответствующей кнопкой командной панели «Установить основным», в этом случае она будет подставляться в документы по умолчанию.

Закладка Коды:

На закладке заполняются вид деятельности и коды организации.

  • ОКПО - Общегосударственный Классификатор Предприятий Организаций.
  • ОКЭД - Общегосударственный Классификатор Экономической Деятельности.

Закладка Прочее:
  • Контрагент - элемент справочника Контрагенты, который идентифицирует организацию при расчетах с ней в филиалах.
  • Головная организация – для филиалов, представительств и других подразделений, не являющихся самостоятельными юридическими лицами, в данном реквизите указывается головная (материнская) организация.

Для элементов справочника можно задать дополнительные аналитические признаки:

  • Значения Свойств - могут выбираться пользователем из справочников, перечислений и т.д. Одной позиции можно присвоить множество значений свойств. Список свойств, присвоенных позиции справочника можно просмотреть и отредактировать, нажав на кнопку «Открыть свойства (Ctrl+Q)» в списке справочника.
  • Значения Категорий - предусмотрена возможность присвоения элементам справочника определенных категорий. Список категорий, присвоенных элементу справочника можно просмотреть и отредактировать, нажав на кнопку «Открыть категории (Ctrl+W)» в списке справочника. Категории в отличие от свойств могут иметь одно из двух значений, точнее обладать или не обладать каким-то свойством.
