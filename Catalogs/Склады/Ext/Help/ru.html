Справочник "Склады (места хранения)"


Справочник "Склады (места хранения)" предназначен для хранения списка всех складов предприятия. В качестве элементов справочника может быть не только список складов, но и список помещений, ангаров и т.д.

Элементы справочника "Склады (места хранения)" используются практически во всех документах, регистрирующих движение товаров, и определяют место хранения, по которому фактически совершено то или иное движение.

В настройках параметров учета на закладке "Аналитический учет ТМЗ" можно установить признаки "Вести складской учет" и "Вести суммовой учета по складам" для бухгалтерского и налогового учета. Признак "Вести складской учет" указывает на то, что учет товаров будет вестись в разрезе складов и на счета учета товарно-материальных ценностей добавится аналитика "Склады", выбор которой будет осуществляться из справочника "Склады (места хранения)". Признак "Вести суммовой учет по складам" указывает на то, что учет товарно-материальных ценностей в разрезе складов будет вестись в количественно-суммовом выражении. Это позволит рассчитывать себестоимость товаров в пределах каждого склада, а не в пределах организации в целом. При таком варианте себестоимость одной и той же номенклатурной позиции на разных складах может отличаться.

В справочник может быть введено любое количество складов. Для удобного разделения списка складов есть возможность создать для них различные группы (например, если на предприятии несколько организаций, то группы помогут разделить склады в пределах каждой организации или обособленного подразделения).

Для каждого склада существует возможность указания или изменения материально-ответственного лица (кладовщика, заведующего складом), которое отвечает за работу данного склада. Выбор ответственного за склад осуществляется из справочника "Физические лица". Список всех материально-ответственных лиц хранится в регистре сведений "Ответственные лица". Просмотреть список ответственных любого склада можно с помощью кнопки "История".

Для каждого склада можно указывать "Префикс". Если префикс указан, то при оформлении документов, в которых указывается склад, номер документа будет задаваться с учетом префикса и организации, и склада (например, для организации установлен префикс "ТД", а для склада – "СК", номер документа будет с префиксом "ТДСК").

Один из складов следует указать в качестве основного склада в настройках пользователя в группе "Основные значения для подстановки в документы и справочники". Указанный склад будет использоваться при начальной установке значений для новых документов.

На панели навигации располагаются ссылки, которые позволяют оперативно просмотреть дополнительную информацию:

  • Регистр сведений "Настройка прав доступа пользователей".
  • Регистр сведений "Ответственные лица".
  • Регистр сведений "Права доступа пользователей".
