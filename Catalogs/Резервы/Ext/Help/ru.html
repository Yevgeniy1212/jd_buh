Справочник "Резервы"

Справочник "Резервы" предназначен для хранения информации о видах формируемых резервов в бухгалтерском учете.

Реквизит "Вид резерва" предназначен для указания того, под какие затраты в будущем создается резерв:

  • По сомнительным требованиям;
  • Под обесценивание ценных бумаг;
  • По вознаграждениям сотрудникам;
  • На покрытие иных затрат.


Для резервов по вознаграждениям сотрудникам в табличной части "Базовые расчеты" необходимо задать список видов начислений заработной платы, которые будут выступать в качестве базы для расчета ежемесячных отчислений в резерв.

Для резервов по вознаграждениям сотрудникам в табличной части "Размеры отчислений" необходимо указать организацию и размеры отчислений в резервы. Если при формировании резерва не требуется учитывать социальный налог и/или социальные отчисления, то соответствующий реквизит следует оставить пустым. Сведения, указанные на данной закладке, автоматически записываются в регистр сведений "Размеры отчислений в резерв".
