Справочник «Номенклатура»

Справочник «Номенклатура» предназначен для хранения информации о товарно-материальных ценностях: товарах, материалах, продукции, комплектах, наборах и прочего.

Данные справочника используются для регистрации движения товаров (поступление, перемещение и выбытие), что позволяет вести учет номенклатурных позиций в пределах организации или склада. Справочник «Номенклатура» для ряда счетов учета товарно-материальных ценностей в плане счетов бухгалтерского и налогового учета установлен в качестве вида субконто. Например, для счетов бухгалтерского учета 1310 «Сырье и материалы», 1320 «Готовая продукция», 1330 «Товары» и т.д.

В справочник может быть введено любое количество номенклатур. Для удобного разделения списка номенклатур целесообразно создавать для них различные группы («Материалы», «Услуги», «Товары» и т.д.). Для каждой группы элементов можно создать подгруппу.

Например, в группе «Материалы» может быть определен ряд подгрупп: «Сырье», «Стройматериалы», «Прочие». Конкретные позиции номенклатуры при этом включены в состав соответствующих подгрупп.

В качестве элементов номенклатур можно указывать различные услуги, для этого предназначен признак «Услуга» на форме элемента.

Для элемента справочника «Номенклатура» обязательными для заполнения являются поля «Наименование» и «Единица измерения». Если указанные поля не заполнены, система не позволит записать элемент справочника. При заполнении наименования элемента справочника автоматически заполняется поле «Полное наименование», которое при необходимости можно изменить. Значение единицы измерения выбирается из справочника «Классификатор единиц измерения».

Дополнительной классификацией элементов справочника является «Вид номенклатуры». Выбор осуществляется из справочника «Виды ТМЗ».

Для каждого объекта номенклатуры можно устанавливать дополнительные свойства, которые в дальнейшем будут использоваться в качестве значений по умолчанию при подстановках в реквизиты документов. Такие свойства можно переопределять в любой момент.

  • «Ставка НДС» - выбирается ставка из одноименного справочника «Ставки НДС», которая по умолчанию будет подставляться во все документы движений ТМЗ и услуг;
  • «Вид НДС при импорте» - используется по умолчанию для подстановки в документ «ГТД по импорту»;
  • «Вид подакцизного ТМЗ» - выбирается вид подакцизного товара;
  • В качестве значения «Ставка акциза» указывается ставка налога за единицу измерения налоговой базы. Выбор осуществляется из справочника «Ставки акциза»;
  • «Коэффициент расчета облагаемой базы акциза» используется для расчета суммы акциза;
  • В поле «Номенклатурная группа» из одноименного справочника указывается номенклатурная группа, в разрезе которой предполагается ведение укрупненного учета.

На панели навигации элемента формы располагается ряд ссылок, которые позволяют просмотреть или отредактировать дополнительную информацию о номенклатуре:

  • Регистр сведений «Встречный выпуск продукции». Содержит правила расчета себестоимости встречного выпуска продукции;
  • Регистр сведений «Основные спецификации номенклатуры». Показывает основные спецификации по данной номенклатуре;
  • Регистр сведений «Счета учета номенклатуры». Позволяет устанавливать и просматривать счета учета номенклатуры, которые будут автоматически подставляться системой в документы, регистрирующие операции в номенклатурой;
  • Регистр сведений «Цены номенклатуры». Показывает актуальную цену номенклатурной позиции на определенную дату;
  • Справочник «Спецификации номенклатуры». Позволяет создавать и просматривать спецификации данной номенклатуры;

В поле «Комментарий» можно ввести любую дополнительную информацию. Весь текст комментария можно просмотреть или отредактировать, нажав на кнопку «Выбрать».
