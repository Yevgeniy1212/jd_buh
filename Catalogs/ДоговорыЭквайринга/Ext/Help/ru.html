Договоры эквайринга

Справочник "Договоры эквайринга"

Справочник вызывается для ввода и редактирования информации из пункта «Главное меню» / «Все функции» / «Справочники» / «Договоры эквайринга».

Эквайринг - группа процессов, связанных с оплатой товаров, работ (услуг) платежными банковскими картами, осуществляется уполномоченным банком-эквайером, путем установки на предприятиях торговли (услуг) платёжных терминалов (POS-терминалов) или импринтеров.

Справочник используется для вычисления тех условий, согласно с которым будут обслуживаться банком операции по оформлению платы клиентов при помощи платежных карт. В договорах эквайринга также регистрируется перечень тех эквайринговых терминалов, при помощи которых будут обслуживаться операции с платежными картами.

Справочник Договоры эквайринга применяется во время оформления операций по оплате заказчиком платежными картами (эквайринговые операции).

Реквизиты:

Реквизиты:
  • Наименование - задается пользователем самостоятельно в зависимости от используемого в организации подхода.
  • Код – заполняется автоматически при записи элемента справочника, а также может быть откорректирован вручную.
  • Эквайрер - из справочника "Контрагенты" выбирается банк, с которым заключен договор эквайринга.
  • Договор - указывается договор из справочника "Договоры контрагентов", по которому будут оформляться взаиморасчеты с банком-эквайрером; договор должен иметь вид договора "Прочее".

Тарифы за расчетное обслуживание:

Табличная часть предназначена для указания процентов торговой уступки, которые зависят от применяемых платежных систем. Процент торговой уступки - это процент комиссии, которую получает банк за предоставление услуг.

Например: если покупатель расплатился платежной картой "VISA International", то предоставленная банком услуга должна быть оплачена в размере % от суммы покупки (суммы, оплаченной картой), установленного Банком-Эквайером, а если покупатель расплатился платежной картой "Eurocard/MasterCard", то соответственно, оплата банку составит % от суммы покупки (суммы, оплаченной картой), установленный уполномоченным банком для этого вида банковской карты.
