Сведения о вычетах ИПН

по гиперссылке Сведения о вычетах ИПН, доступно оформление Заявления на предоставление вычетов ИПН.

Документ предназначен для регистрации права предоставления налоговых вычетов по индивидуальному подоходному налогу в соответствии со статьей 343 Налогового Кодекса РК.

В шапке документа заполняются следующие сведения:

  • Организация
  • Структурное подразделение
  • Сотрудник
  • Месяц
  • Действует по

Применение вычетов будет осуществляться по периоду указанному в поле Месяц. Начало действия и прекращения применения вычетов устанавливается пользователем. Если будет не заполнено поле Действует по, то вычеты будут применяться без ограничения. Вычеты, для которых предусмотрен график платежей, применяются согласно данных графика и периода его действия.
В табличной части документа заполняются сведения о предоставленных видах вычетов:

  • Вид вычета
  • Установление признака – Предоставлять вычет
  • Документ, подтверждающий право на вычет.

Обязательным условием для проведения документа является указание признака наличие заявления.
