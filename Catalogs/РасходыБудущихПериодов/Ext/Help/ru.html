Справочник "Расходы будущих периодов"

Справочник вызывается для ввода и редактирования информации из пункта «Главное меню» / «Все функции» / «Справочники» / «Расходы будущих периодов».

Справочник «Расходы будущих периодов» предназначен для хранения информации о расходах, которые осуществлены в текущий момент, но связаны с получением дохода в будущем.

Используется в бухгалтерском и налоговом учете.

Содержит реквизиты, необходимые для автоматизированного списания расходов будущих периодов в различных видах учета.

Реквизиты:
  • Наименование - задается пользователем самостоятельно в зависимости от используемого в организации подхода.
  • Код – заполняется автоматически при записи элемента справочника, а также может быть откорректирован вручную.
  • Контрагент
    • - поставщик, у которого приобретены активы, услуги, классифицированные как расходы будущих периодов.
  • Способ списания - способ списания РБП:
    • Равномерно. При выполнении регламентной операции списания РБП документом "Закрытие месяца" на затраты будет отнесена величина, рассчитываемая как отношение суммы, указанной для элемента РБП в справочнике к количеству месяцев использования актива.
    • От остаточной стоимости. Сумма, относимая на затраты при выполнении регламентной операции списания РБП документом "Закрытие месяца", будет рассчитываться исходя из остаточной суммы РБП по данным бухгалтерского учета по отношению к оставшемуся количеству месяцев использования актива.
    • Не рассчитывать. При выполнении регламентной операции списания РБП документом "Закрытие месяца" данный элемент РБП не будет учитываться.
  • Сумма - величина расхода будущих периодов.
  • Дата начала списания, Дата окончания списания - показатели, задающие временной период списания расхода.

Закладка Бухгалтерский учет:
  • Счет БУ, Субконто БУ 1...Субконто БУ 3 - счет и аналитика, на которые будут списываться расходы будущих периодов в бухгалтерском учете.

Закладка Налоговый учет:
  • Счет НУ, Субконто НУ 1...Субконто НУ 3 - счет и аналитика, на которые будут списываться расходы будущих периодов в налоговом учете.

Элементы справочника могут быть сгруппированы в группы.
