Подразделения организаций

Справочник "Подразделения организаций"

Справочник вызывается для ввода и редактирования информации из пункта «Главное меню» / «Все функции» / «Справочники» / «Подразделения организаций».

Справочник «Подразделения организаций» используется для ведения аналитического учета в разрезе подразделений каждой организации.

Справочник позволяет вести учет затрат на местах производства работ (цех №1, цех №2) и по местам концентрации хозяйственных функций (дирекция, бухгалтерия).

Подразделения организаций также используются в ряде документов по учету основных средств.

Реквизит Отражение зарплаты в учете:

Реквизиты:
  • Наименование - задается пользователем самостоятельно в зависимости от используемого в организации подхода.
  • Код – заполняется автоматически при записи элемента справочника, а также может быть откорректирован вручную.
  • Признак «Является структурным подразделением» - следует установить, если в целях налогового учета информация по подразделению должна предоставляться отдельно от данных других подразделений.

Формирование налоговой отчетности в разрезе структурных подразделений осуществляется только в случае включения соответствующей поддержки в настройках параметров учета. Изменение параметров учета необходимо производить до начала ведения учета в информационной базе, в противном случае требуется перепроведения всех существующих документов для корректного разделения сведений разных структурных подразделений.

При установленном признаке «Является структурным подразделением», отображаются для заполнения следующие реквизиты.

Закладка Основные:
  • Указывать реквизиты головной организации в счете фактуре - позволяет организовать автоматическую подстановку значения головной организации в качестве поставщика/покупателя в счете фактуре.
  • Полное наименование – для размещения полного официального наименования структурного подразделения.
  • БИН / ИИН - бизнес - идентификационный номер (для юридического лица) / индивидуальный идентификационный номер (для физического лица).
  • Налоговый комитет – Налоговый комитет по месту регистрации структурного подразделения.
  • Относить налоги к бюджету городов районного значения, сел, поселков, сельских округов - если данный признак установлен, то для видимости будет доступен выбор Аппарата акимата для форм налоговой отчетности.

Закладка Контакты:

В таблице представлена контактная информация структурного подразделения.

Закладка Банковские счета:

В таблице «Банковские счета» отображаются все банковские счета структурного подразделения.

Списки банковских счетов могут дополняться.

Жирным шрифтом отмечаются основной расчетный счет, который можно изменить соответствующей кнопкой командной панели «Установить основным».

Закладка Кассы:

В таблице «Кассы» отображаются все кассы структурного подразделения.

Список касс может дополняться.

Жирным шрифтом отмечается основная касса, которую можно изменить соответствующей кнопкой командной панели «Установить основной», в этом случае она будет подставляться в документы по умолчанию.

Закладка Коды:

На закладке заполняются вид деятельности и коды организации.

  • ОКПО - Общегосударственный Классификатор Предприятий Организаций.
  • ОКЭД - Общегосударственный Классификатор Экономической Деятельности.



Реквизит Отражение зарплаты в учете:

В разделе можно указывать правило отражения расчетов по заработной плате с сотрудниками подразделения в бухгалтерском и налоговом учете. Подробный порядок настройки способов отражения приводится в справочной информации к документу "Отражение зарплаты в регл. учете".


Для элементов справочника можно задать дополнительные аналитические признаки:



Элементы справочника могут быть сгруппированы по группам.

  • Значения Свойств - могут выбираться пользователем из справочников, перечислений и т.д. Одной позиции можно присвоить множество значений свойств. Список свойств, присвоенных позиции справочника можно просмотреть и отредактировать, нажав на кнопку «Открыть свойства (Ctrl+Q)» в списке справочника.
  • Значения Категорий - предусмотрена возможность присвоения элементам справочника определенных категорий. Список категорий, присвоенных элементу справочника можно просмотреть и отредактировать, нажав на кнопку «Открыть категории (Ctrl+W)» в списке справочника. Категории в отличие от свойств могут иметь одно из двух значений, точнее обладать или не обладать каким-то свойством.
