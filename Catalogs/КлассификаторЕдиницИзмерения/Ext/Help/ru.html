Справочник «Классификатор единиц измерения»

Справочник «Классификатор единиц измерения» предназначен для хранения списка единиц измерения, используемых организацией. Единица измерения указывается, например, при вводе номенклатуры.

Список единиц измерения можно заполнять автоматически из классификатора «Общий классификатор единиц измерения», который открывается по кнопке «Подбор» из формы списка элементов справочника. В открывшемся списке выбирается необходимая единица измерения и двойным щелчком мыши или нажатием кнопки «Enter» добавляется в справочник. При выборе значения из классификатора открывается форма элемента, в которой поля «Код», «Полное наименование» и «Краткое наименование» единицы измерения заполняются автоматически. При необходимости эти значения можно изменить. Если такой элемент уже есть в справочнике, то конфигурация выдаст предупреждающее сообщение, если такого элемента нет, то будет сформирована новая форма элемента с заполненными параметрами. При нажатии на кнопку «Записать и закрыть» эта информация добавится в справочник.

Для элемента справочника обязательным для заполнения является поле «Наименование». Если указанное поле не заполнено, система не позволит записать элемент справочника.
