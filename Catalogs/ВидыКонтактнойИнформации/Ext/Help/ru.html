Справочник «Виды контактной информации»

Справочник «Виды контактной информации» предназначен для хранения классификации контактной информации различных объектов системы: организаций, физических лиц и т.д.

Поле «Тип» предназначено для определения вида контактной информации из заложенного в системе фиксированного списка:

  • Адрес – реквизит с таким типом контактной информации предназначен для хранения адреса в виде произвольной строки или в виде шаблона адреса. Позволяет заполнять адрес, используя адресный классификатор. В таком случае индекс адреса устанавливается из классификатора автоматически после окончания ввода адреса. Предопределенные значения фактического или юридического адреса используются в печатных формах документов;
  • Телефон – реквизит с таким типом контактной информации предназначен для хранения телефонных номеров в виде произвольной строки или в виде шаблона ввода номера телефона. Предопределенное значение телефона используется в печатных формах документов;
  • E-mail – реквизит с таким типом контактной информации предназначен для хранения адреса электронной почты. Реквизит используется при отправке электронных писем из информационной базы;
  • Веб-страница – реквизит с таким типом контактной информации предназначен для хранения адресов веб страниц;
  • Другое – реквизит с таким типом контактной информации предназначен для хранения произвольной текстовой контактной информации.

Поле «Вид объекта контактной информации» предназначено для указания объекта, для которого будет использоваться данный тип контактной информации. Список видов объектов контактной информации зафиксирован в системе:

  • Справочник «Контактные лица»;
  • Справочник «Контрагенты»;
  • Справочник «Организации»;
  • Справочник «Пользователи»;
  • Справочник «Физические лица»;
  • Справочник «Подразделения организации».

Например, для справочника «Организации» с типом контактной информации Адрес можно определить значения «Юридический адрес организации» или «Фактический адрес организации». А для справочника «Физические лица» с таким же типом контактной информации – «Адрес», можно определить значения «Адрес по прописке физ. лица» или «Адрес проживания физ. лица». Например, если в справочнике «Организации» на закладке «Контактная информация» указать значения для типов контактной информации «Адрес» и «Телефон», эти значения будут использоваться в печатных формах документов и отчетах.

Некоторые элементы справочника являются предопределенными и не могут редактироваться пользователем.

Все поля элемента справочника являются обязательными для заполнения. Если поля не заполнены, система не позволит записать элемент справочника.
