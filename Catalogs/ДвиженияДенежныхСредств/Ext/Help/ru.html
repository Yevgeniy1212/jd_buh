Справочник «Движения денежных средств»

Справочник «Движения денежных средств» предназначен для хранения информации о видах движения денежных средств. Элементы справочника используются в качестве реквизита справочника «Статьи движения денежных средств».

На основании данных справочника заполняется форма регламентированного отчета «Отчет о движении денег».

Справочник является иерархическим, что позволяет пользователю разделять элементы по группам. Для элемента справочника обязательным для заполнения является поле «Наименование». Если указанное поле не заполнено, система не позволит записать элемент справочника.
