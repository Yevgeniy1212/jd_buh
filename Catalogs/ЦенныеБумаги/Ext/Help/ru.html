Справочник «Ценные бумаги»

Данный справочник предназначен для ведения списка ценных бумаг, принадлежащих организации на праве собственности, и используется для ведения аналитического учета на счетах 1120 «Краткосрочные финансовые активы, оцениваемые по справедливой стоимости через прочий совокупный доход», 1130 «Краткосрочные финансовые активы, оцениваемые по справедливой стоимости через прибыль или убыток», 2020 «Долгосрочные финансовые активы, оцениваемые по справедливой стоимости через прочий совокупный доход», 2030 «Долгосрочные финансовые активы, оцениваемые по справедливой стоимости через прибыль или убыток» бухгалтерского учета, 3020 «Краткосрочные финансовые обязательства, оцениваемые по справедливой стоимости через прибыль или убыток»,  3070 «Текущая часть долгосрочных финансовых обязательств, оцениваемых по справедливой стоимости через прибыль или убытки» и 4020 «Долгосрочные финансовые обязательства, оцениваемые по справедливой стоимости через прибыль или убыток».

Элементами справочника являются конкретные виды ценных бумаг (например, Вексель).
