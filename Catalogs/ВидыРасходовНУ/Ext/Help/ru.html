Справочник «Виды расходов в налоговом учете»

Справочник «Виды расходов в налоговом учете» предназначен для хранения категорий расходов, определенных в налоговом учете. Для расходов, относимых на вычеты по корпоративному подоходному налогу в пределах установленных законодательством норм, необходимо установить признак Нормируемый. В этом случае, при формировании движений по налоговому учету первичными документами, корреспонденции будут формироваться с видом учета НУ «ПР» («Постоянная разница»). В конце периода в результате расчетов по налогу на прибыль те расходы, которые не превысили законодательно установленные пределы, будут отнесены на вычеты.

Проведение документов программы будет зависеть от установок реквизитов этого справочника. Поэтому не рекомендуется менять значение реквизитов, после того как элемент справочника был использован в проведенных документах. В противном случае документы необходимо перепровести.

Справочник является иерархическим, что позволяет пользователю разделять элементы по группам. Для элемента справочника обязательным для заполнения является поле «Наименование». Если указанное поле не заполнено, система не позволит записать элемент справочника.
