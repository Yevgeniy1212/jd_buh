Справочник «Операции по движению подакцизных ТМЗ»

Справочник «Операции по движению подакцизных ТМЗ» предназначен для ведения списка операций по движению подакцизных товаров и материалов, в соответствии с классификацией операций в Декларации по акцизу. Учет операций по движению операций подакцизных товаров ведется для организаций, являющихся плательщиками акциза. Данная настройка устанавливается в регистре сведений «Учетная политика (по налоговому учету)».

Элементами справочника являются конкретные виды операций по виду подакцизных товаров, например, «Реализация», «Порча, утрата» и т. д.

Для элемента справочника обязательным для заполнения является поле «Наименование». Если указанное поле не заполнено, система не позволит записать элемент справочника.
