Справочник "Биологические активы"

Справочник "Биологические активы" предназначен для хранения списка имеющихся в организации биологических активов. Для ряда счетов в плане счетов бухгалтерского учета справочник "Биологические активы" установлен в качестве вида субконто:

  • 2510 «Растения»;
  • 2520 «Животные».


Для каждого элемента справочника задаются значения реквизитов:

  • Наименование. Реквизит является обязательным для заполнения.
  • Код КОФ. Код по классификатору основных фондов. Выбор осуществляется из справочника "Классификатор основных фондов".


На панели навигации элемента формы располагается ссылка, которая позволяет просмотреть дополнительную информацию:

  • Регистр сведений "Объекты имущественного налога". Содержит информацию о признаке обложения налогом на имущество основных средств.
