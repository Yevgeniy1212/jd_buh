Справочник "Базы распределения производственных затрат"

Справочник добавлен в типовую конфигурацию.

Справочник "Базы распределения производственных затрат" предназначен для хранения правил, по которым возможно распределение производственных затрат.

При заполнении справочника следует придерживаться следующих правил:

-         В справочнике должен обязательно присутствовать элемент с базой распределения «по умолчанию», т.е. элемент используемый, «как правило», для большинства случаев;

-         Должны быть указаны возможные исключения из правила по конкретным статьям затрат и по подразделению;

-         Подразделение должно быть обязательно заполнено (элемент или группа любого уровня).

Способ распределения производственных затрат может быть установлен для всех уровней иерархии справочника «Подразделения организации» и справочника «Статьи затрат». Например, способ распределения производственных затрат может быть установлен:

-         для месторождения (участка) в целом (группа элементов справочника),

-         для конкретной бригады, подразделения.

В документах, проводящих распределение затрат согласно действующей методике, поиск базы распределения в справочнике осуществляется на основании информации о подразделении и статье затрат документа-первоисточника следующим образом:

шаг 1.      Ищется элемент по соответствию (или входящим в группу) с выбранным в документе подразделением. Например, для указанной в документе бригады может быть установлена (и найдена) в справочнике база распределения по участку, включающему эту бригаду.

шаг 2.      Далее ищется элемент по соответствию (или входящим в группу) с выбранной в документе статьей затрат. Если статья затрат не найдена – используется база для указанного подразделения (шаг 1) с пустой статьей затрат.

Например, база распределения для строки, в которой подразделение указано как  «Бурение», будет использована для всех бригад, входящих в эту группу, т.к. элемент «Бурение» является группой.

Алгоритм распределения в соответствии с базами

Во всех документа, где используется распределение в соответствие с выбранной базой, происходят следующие расчеты:

1.      Если в документе указано подразделение с видом «Бригада», то распределение затрат будет производиться по скважинам указанной бригады в соответствии с производственной справкой и показателем распределения;

2.      Если в документе указано подразделение с видом «Участок», то распределение затрат будет производиться по скважинам всех бригад указанного участка в соответствии с производственной справкой и показателем распределения;

3.      Если в документе указано подразделение с видом «Подразделение», то распределение затрат будет производиться по скважинам всего ГРЭ в соответствии с производственной справкой и показателем распределения;

4.      Если в документе указана база распределения со способом распределения «по часам освоения», то выбранное подразделение должно быть указано в соответствующем участке как подразделение освоения . Распределение затрат будет производиться по скважинам участка, к которому относится это подразделение в соответствии с производственной справкой и показателем распределения.
