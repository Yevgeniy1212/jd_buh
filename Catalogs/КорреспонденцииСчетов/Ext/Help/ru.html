Справочник "Корреспонденции счетов"

Справочник вызывается для ввода и редактирования информации из пункта «Главное меню» / «Все функции» / «Справочники» / «Корреспонденции счетов».

Справочник «Корреспонденции счетов» содержит набор стандартных корреспонденций счетов бухгалтерского учета.

Поиск корреспонденции в справочнике возможен по счетам (дебета, кредита или обеих сторон корреспонденции), а также по содержанию проводки (ее отдельным словам, фрагментам слов), либо по документу.
