Справочник «Документы удостоверяющие личность»

Справочник «Документы удостоверяющие личность» предназначен для хранения перечня видов документов, удостоверяющих личность. Для каждого физического лица можно указать реквизиты документа, удостоверяющего его личность. Для указания вида документа и используется этот справочник.

Справочник может быть заполнен при помощи классификатора «Виды документов, удостоверяющих личность», для этого в форме списка справочника предназначена кнопка «Подбор». Если при выборе значения из классификатора такой элемент уже есть в справочнике, то конфигурация выдаст предупреждающее сообщение, если такого элемента нет, то будет сформирована новая форма элемента с заполненными параметрами.

Реквизит «Код вида документа в налоговой отчетности» необходимо заполнить в случае, если документ удостоверяет личность лица, не являющегося гражданином Республики Казахстан. Реквизит заполняется из зафиксированного в системе списка:

  • 01 – Паспорт иностранного гражданина;
  • 02 – Удостоверение личности иностранного гражданина;
  • 03 – Паспорт моряка;
  • 04 – Вид на жительство.

Значение данного реквизита используется при заполнении графы G «Вид, номер и дата выдачи документа, удостоверяющего личность» приложения 210.01 регламентированного отчета «Форма 210.00. Декларация по индивидуальному подоходному налогу и социальному налогу по иностранцам и лицам без гражданства».

Для элемента справочника обязательным для заполнения является поле «Наименование». Если указанное поле не заполнено, система не позволит записать элемент справочника.
