Справочник «Категории земель»

Справочник «Категории земель» предназначен для хранения информации о категориях земель в соответствии с земельным законодательством Республики Казахстан.

Ввод новых элементов справочника может осуществлять вручную или с помощью классификатора «Классификатор категорий земель в соответствии с земельным законодательством». Для добавления нового элемента справочника из классификатора предназначена кнопка «Подбор».

В поле «Код по классификатору» указывается код строки.

В поле «Вид земельного налога» указывается элемент справочника «Налоги, сборы, отчисления».
