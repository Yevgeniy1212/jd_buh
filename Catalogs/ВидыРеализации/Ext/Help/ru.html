Справочник «Виды реализации ТМЗ (работ, услуг)»

Справочник содержит виды реализации ТМЗ для целей НДС.

У вида реализации заполняется наименование (краткое, полное). Если вид реализации является корректировкой оборота по реализации, то устанавливается признак Корректировка оборота по реализации. В разделе «Отражение декларации по НДС» заполняются: период, вид реализации и соответствующая строка в декларации.

Например: Вид поступления – Возврат ТМЗ. Данный вид является корректировкой оборота по реализации, поэтому в справочнике устанавливается признак. В разделе «Отражение декларации по НДС» заполняется период 01.01.14, вид поступления - Возврат ТМЗ и код строки – 300.06.001

Для элемента справочника обязательным для заполнения является поле «Наименование». Если указанное поле не заполнено, система не позволит записать элемент справочника.
