Справочник «Валюты»

Данный справочник используется во всех видах учета (бухгалтерском и налоговом).

В справочнике валют содержится перечень валют, которыми организация пользуется в своей деятельности: тенге, российский рубль, доллар США и т.д. По умолчанию в качестве валюты регламентированного учета (базовой валюты) в конфигурации установлен тенге.

Справочник валют вызывается для ввода и редактирования информации из пункта меню «Справочники» – «Валюты».

Ввод новой валюты в справочник валют

Новую валюту можно ввести путем добавления информации в справочник «Валюты». Для этого необходимо воспользоваться специальной кнопкой «Создать». При создании новой валюты заполняются поля: код валюты, наименование, полное наименование, параметры прописи на русском.

Для того чтобы в формах печати правильно формировалось наименование валюты прописью, необходимо задать параметры прописи на русском.

Для того чтобы посмотреть историю курсов валют и их кратности, необходимо на верхней панели справочника выбрать Курсы валют. После выбора, откроется регистр сведений «Курсы валюты», который предназначен для хранения информации об истории курсов валют и их кратности, и используется в учете по счетам с признаком валютного учета.
