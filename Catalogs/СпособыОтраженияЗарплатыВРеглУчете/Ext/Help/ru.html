Справочник "Способы отражения зарплаты в регламентированном учете"

Справочник предназначен для хранения способов отражения заработной платы в регламентированном учете. Способы отражения зарплаты в учете представляют собой шаблоны корреспонденции счетов, которые будут сформированы при отражении начисленной зарплаты сотрудников организации в бухгалтерском и налоговом учете.

Способы "Не отражать в бухучете" и "Отражение начислений по умолчанию" являются предопределенными и не могут быть удалены. Если для строки начисления не был найден ни один подходящий способ отражения в соответствии с настройками, то будет использоваться способ "Отражение начислений по умолчанию".

При вводе способа отражения зарплаты в учете нужно указывать:

  • Наименование. Наименование предназначено для идентификации того или иного способа отражения. В наименовании рекомендуется указывать корреспонденцию счетов для более удобного использования при назначении способа отражения в учете.
  • Счет Дт. Указывается счет дебета корреспонденции счетов и аналитика.
  • Счет Кт. Указывается счет кредита корреспонденции счетов и аналитика.


При описании способов отражения зарплаты в регламентированном учете для счетов учета допускается не указывать некоторые субконто. В этом случае программа сама подберет их значения автоматически.

  • Значение субконто "Подразделение организации". На затратных счетах значение субконто будет подобрано автоматически по подразделению, в котором числится сотрудник.
  • Значение субконто "Работники организации". На счетах по оплате труда значение субконто будет подобрано автоматически по работникам, расчеты которых отражаются в учете.
  • Значение субконто "Контрагенты". На счете 3385 "Задолженность по исполнительным листам" при отражении удержаний по исполнительным листам значение субконто будет подобрано автоматически по данным о получателе, указанном в документе "Исполнительный лист".


Способ отражения зарплаты в регламентированном учете может быть задан несколькими способами:

  • Способ отражения, заданный в документе по расчету начислений и удержаний.
  • Способ отражения, заданный с помощью документа "Ввод сведений о регл. учете плановых начислений сотрудников организации".
  • Способ отражения, заданный на закладке "Учет" в окне настройки вида начисления.
  • Способ отражения, заданный на закладке "Отражение зарплаты в учете" элемента справочника "Подразделения организации".
