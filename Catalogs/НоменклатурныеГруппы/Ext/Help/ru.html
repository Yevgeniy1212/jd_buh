Справочник «Номенклатурные группы»

Справочник «Номенклатурные группы» предназначен для хранения перечня номенклатурных групп, в разрезе которых ведется укрупненный учет затрат основного и вспомогательного производства, а также выручки, полученной от реализации товаров, продукции, услуг.

В составе номенклатурной группы указывается список номенклатурных позиций (товаров, продукции, услуг).

Например, в состав номенклатурной группы «Предоставление услуг» может входить весь перечень услуг, оказываемых организацией. В состав номенклатурной группы «Производство выпечки» может входить весь перечень выпечки, выпускаемой организацией (булочки, вафли и т.д.).

В состав одной номенклатурной группы может входить неограниченное количество позиций номенклатуры, но одна позиция номенклатуры может входить только в одну номенклатурную группу. Если в номенклатурную группу попытаться добавить позицию номенклатуры, которая уже есть в одной из групп, конфигурация выдаст предупреждение.

На панели навигации элемента формы располагается ряд ссылок, которые позволяют просмотреть или отредактировать дополнительную информацию о номенклатурной позиции:

  • Регистр сведений «Встречный выпуск продукции». Содержит правила расчета себестоимости встречного выпуска продукции;
  • Регистр сведений «Основные спецификации номенклатуры». Показывает основные спецификации по данной номенклатуре;
  • Регистр сведений «Счета учета номенклатуры». Позволяет устанавливать и просматривать счета учета номенклатуры, которые будут автоматически подставляться системой в документы, регистрирующие операции в номенклатурой;
  • Регистр сведений «Цены номенклатуры». Показывает актуальную цену номенклатурной позиции на определенную дату;
  • Справочник «Спецификации номенклатуры». Позволяет создавать и просматривать спецификации данной номенклатуры.

Справочник является иерархическим, что позволяет пользователю разделять элементы по группам. Для элемента справочника обязательным для заполнения является поле «Наименование». Если указанное поле не заполнено, система не позволит записать элемент справочника.
