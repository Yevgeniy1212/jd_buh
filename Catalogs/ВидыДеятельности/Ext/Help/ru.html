Справочник «Виды деятельности»

Справочник «Виды деятельности» предназначен для хранения перечня видов деятельности организации. Каждый вид деятельности характеризуется уникальным кодом и наименованием. Элементы справочника используются для указания вида деятельности организации (элемент справочника «Организации»).

Для каждого элемента справочника указывается код вида деятельности из классификатора «Общий классификатор видов экономической деятельности». В зависимости от выбранного кода автоматически устанавливается полное наименование вида деятельности.

Справочник является иерархическим, что позволяет пользователю разделять элементы по группам. Для элемента справочника обязательным для заполнения является поле «Наименование». Если указанное поле не заполнено, система не позволит записать элемент справочника.
