Справочник «Нематериальные активы»

Справочник «Нематериальные активы» предназначен для хранения списка и наиболее общей информации о нематериальных активах.

Для элемента справочника можно определить дополнительные свойства:

  • Вид НМА – выбирается вид нематериального актива:
    • Гудвилл;
    • Организационные расходы;
    • Лицензионное соглашение;
    • Программное обеспечение;
    • Патенты;
    • Оригиналы литературных и художественных произведений;
    • Прочие нематериальные активы;
    • Ноу хау.
  • Код КОФ – указывается код по классификатору основных фондов;
  • Прочие сведения.

Данные справочник используются для регистрации движения нематериальных активов, что позволяет вести их учет в пределах организации. Справочник «Нематериальные активы» для ряда счетов учета НМА в планах счетов бухгалтерского и налогового учета установлен в качестве вида субконто. Например, для группы счетов бухгалтерского учета 2700 «Нематериальные активы».

На панели навигации элемента формы располагается ряд ссылок, которые позволяют просмотреть или отредактировать дополнительную информацию о нематериальном активе:

  • Регистр сведений «Объекты имущественного налога». Содержит информацию о признаке обложения налогом на имущество;
  • Регистр сведений «Первоначальные сведения о НМА». Содержит информацию о первоначальных сведениях нематериальных активов организации для целей бухгалтерского учета;
  • Регистр сведений «Сведения о группах фиксированных активов». Содержится информация о параметрах налогового учета фиксированных активов: предельной норме амортизации, применяемой норме амортизации, проценте для вычета расходов на ремонт;
  • Регистр сведений «Состояние НМА организации». Содержит историю изменений состояний нематериальных активов;
  • Регистр сведений «Счета учета НМА». Позволяет просматривать счета учета и счета учета амортизации в организации объектов НМА;
  • Регистр сведений «Фиксированные активы организации». Содержит сведения о составе групп налогового учета в разрезе фиксированных активов;
  • Регистр сведений «Фиксированные активы, учитываемые отдельно». Содержит сведения о фиксированных активах, для которых в первый год эксплуатации применяется двойная норма амортизации.

Справочник является иерархическим, что позволяет пользователю разделять элементы по группам. Для элемента справочника обязательным для заполнения является поле «Наименование». Если указанное поле не заполнено, система не позволит записать элемент справочника.

Для элемента справочника предусмотрена печатная форма «Форма ДА-2».
