Справочник «Спецификации номенклатуры»

Справочник «Спецификации номенклатуры» предназначен для хранения спецификаций. Спецификации – это перечень материалов (полуфабрикатов), которые используются для изготовления определенного количества единиц готовой продукции.

Для позиции номенклатуры может быть указано несколько спецификаций. В регистре сведений «Основные спецификации номенклатуры» одну из спецификаций номенклатурной позиции можно установить в качестве основной, которая будет использоваться по умолчанию.

В графе «Выходное изделие (продукция, полуфабрикат, услуга)» указывается готовая продукция и ее количество. В табличной части «Исходные комплектующие» указывается список материалов (полуфабрикатов) и их количество, используемое для изготовления указанного количества выходного изделия. Табличная часть может быть заполнена построчным вводом (при помощи кнопки «Добавить») или подбором (при помощи кнопки «Подбор») материалов из справочника «Номенклатура».

Справочник является иерархическим, что позволяет пользователю разделять элементы по группам. Для элемента справочника обязательным для заполнения являются поля «Наименование» и «Номенклатура». Если указанные поля не заполнены, система не позволит записать элемент справочника.

Кнопка «Создать на основании» предназначена для автоматического заполнения документа «Требование-накладная»

На основании данных элемента справочника может быть оформлен документ «Требование-накладная» (кнопка «Создать на основании»). Документ автоматически заполнится данными из элемента справочника.

Для элемента справочника «Спецификации номенклатуры» предусмотрена печатная форма «Спецификация».
