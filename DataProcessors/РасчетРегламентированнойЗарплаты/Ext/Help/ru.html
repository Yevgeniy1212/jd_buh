Инструмент «Расчет регламентированной зарплаты»

Инструмент позволяет создавать, рассчитывать и проводить документы по расчету заработной платы за месяц. В процессе расчета обрабатываются документы следующих видов «Начисление зарплаты сотрудникам организаций», «Расчет удержаний сотрудников организаций», «Расчет налогов, взносов и отчислений сотрудников организации», «Удержание ИПН, ОПВ и ВОСМС по налоговому учету» и «Отражение зарплаты в регл учете». Виды автоматически рассчитываемых документов устанавливаются с помощью флажков, расположенных в форме настройки, вызываемой при нажатии на одноименную кнопку на верхней панели инструментов формы обработки. По умолчанию производится расчет всех указанных видов документов.

Для расчета необходимо выбрать месяц расчета и указать организацию, заработная плата сотрудников которой будет рассчитываться. Расчет производится по каждому отмеченному подразделению из списка подразделений организации, т.е. для каждого отмеченного подразделения создается свой документ каждого типа, если среди уже существующих документов нет документа, сформированного по родительскому подразделению или по всей организации в целом, иначе будет перерассчитываться уже существующий документ. Для этого предназначены пункт меню «Рассчитать» на верхней панели формы.

В процессе расчета осуществляются следующие действия:

  1. Анализируются существующие в информационной базе документы перечисленных видов за указанный месяц по организации, определяются подразделения, по которым отсутствуют какие-либо из указанных документов. При необходимости недостающие документы будут созданы. Документы «Начисление зарплаты сотрудникам организаций» и «Расчет удержаний сотрудников организаций», у которых заполнен реквизит «Вид расчета» (т.е. при заполнении осуществляется отбор только по определенному виду расчета), при определении документов, подлежащих расчету, не учитываются, т.к. не содержат полный список всех плановых расчетов, поэтому в табличном поле документов не отображаются. Кроме того, в процессе анализа игнорируются документы «Начисление зарплаты сотрудникам организаций», содержащие виды расчета без признака «Зачет отработанного времени», поскольку такие виды расчета рассматриваются как разовые и неподлежащие ежемесячному расчету.
  2. Для каждого документа последовательно вызываются функции автозаполнения документа, расчета и проведения. Документы обрабатываются в следующем порядке:
    • Начисление зарплаты сотрудникам организаций;
    • Расчет удержаний сотрудников организаций;
    • Расчет налогов, взносов и отчислений сотрудников организации;
    • Удержание ИПН, ОПВ и ВОСМС по налоговому учету (если в учетной политике по налоговому учету для организации не указан признак «При начислении ИПН и ОПВ принимать исчисленные суммы к учету как удержанные», иначе данный вид документов вообще не обрабатывается);
    • Отражение зарплаты в регл учете (формируется только один документ по всей организации за месяц).

Внимание!  Так как для учета возможных изменений в кадровых документах по сотрудникам организации в процессе расчета всегда выполняется автозаполнение документов перед расчетом, следует избегать ручных корректировок документов либо отключить их автоматический расчет.

По результатам расчета возможно формирование следующих отчетов непосредственно из формы обработки:

  • Свод начислений и удержаний;
  • Расчетная ведомость организаций;
  • Расчетный листок;
  • Расчет по исчислению и удержанию индивидуального подоходного налога и исчислению социального налога;
  • Контрольная ведомость по отражению зарплаты в регламентированном учете.

Формирование отчетов осуществляется по выбранной организации за месяц проведения расчета. Возможно формировать отчеты только по отмеченным подразделениям организации - для этого следует отметить флажок «По отмеченным подразделения», либо только по указанному сотруднику - его следует выбрать в соответствующем поле на закладках «Расчетный листок» и «Расчет ИПН и СН».
