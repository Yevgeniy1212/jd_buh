Перед загрузкой выберите:

  • Валюты, для которых будут загружены курсы.
  • Файлы, из которых будут загружены курсы валют.

Затем нажмите кнопку «Загрузить курсы». Загруженные курсы сохраняются в регистре сведений «Курсы валют».

Файл, используемый для загрузки, должен соответсововать формату файла курсов валют Национального Банка РК.

Файл, содержащий курсы валют относительно KZT, на заданную дату, можно скачать по адресу:

  • http://www.nationalbank.kz/rss/get_rates.cfm?fdate=День.Месяц.Год.

Например:

  • http://www.nationalbank.kz/rss/get_rates.cfm?fdate=01.01.2012

 

Для загрузки курсов валют с веб-сайта Национального Банка РК необходимо выбрать команду: Все действия - Загрузить курсы из Интернета...

См. также: Валюты, Курсы валют, Загрузка курсов из Интернета.
