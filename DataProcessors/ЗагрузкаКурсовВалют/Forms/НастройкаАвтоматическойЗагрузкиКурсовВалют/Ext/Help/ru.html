Программа позволяет автоматически загружать актуальные курсы валют с веб-сайта Национального Банка РК.

Автоматическая загрузка курсов происходит для валют, у которых установлен признак: «Загружать курс из Интернета».

Настройка действует для всех пользователей.

Загрузка происходит с помощью регламентного задания.

См. также: Валюты, Курсы валют, Параметры прокси-сервера.
