Печать штрихкодов

Обработка предназначена для групповой печати этикеток штрихкодов. На печать будут выведены этикетки основных средств, выбранных в списке и у которых сгенерирован штрихкод.
Печать штрихкодов производится по шаблону, указанному в обработке. По умолчанию подставляется предопределенный элемент.

Так же оброботка может быть использована для формирования штрихкода у существующих основных средств. В данной обработке с помощью кнопки Подбор нужно выбрать основные средства, для которых требуется сформировать штрихкод, после этого нажать на кнопку Сгенерировать штрихкоды. Штрихкод будет сгенерирован для тех основных средств, которые будут отмечены в списке.
