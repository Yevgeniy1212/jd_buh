Обработка «Выгрузка данных в конфигурацию Зарплата и Управление Персоналом для Казахстана»

В конфигурацию «Зарплата и Управление Персоналом для Казахстана»  могут быть выгружены следующие данные:

  • Данные для бухгалтерского учета:
    • План счетов;
    • Виды субконто;
    • Данные аналитического учета для отмеченных счетов.
  • Данные для налогообложения доходов лиц, у которых приобретаются товары, работы или услуги (акты закупа), а также данные о доходах, выплаченных индивидуальным предпринимателям, адвокатам и частным нотариусам для отражения их в формах 200, 210.
  • Приходные и расходные кассовые ордера, содержащие данные по заработной плате.

Для корректной выгрузки необходимо, чтобы в справочнике «Организации» обеих конфигурации была введена организация, по которой производится выгрузка данных, и для нее была настроена учетная политика.
