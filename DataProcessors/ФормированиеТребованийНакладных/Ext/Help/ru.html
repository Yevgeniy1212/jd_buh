Обработка "Формирование требований-накладных"

Обработка предназначена для формирования требований-накладных отдельно на каждую продукцию, указанную в "Отчете производства за смену".

В поле "Отчет производства за смену" следует указать документ, на основании которого должны быть сформированы требования-накладные, список требований-накладных будет заполнен автоматически. При вызове обработки из документа "Отчет производства за смену" все поля будут заполнены автоматически.

С помощью кнопок "Создать" и "Провести" можно сформировать и провести документы требования-накладные по отмеченной продукции. Действия "создания" и "проведения" документов допускается разнести во времени - при выборе документа в поле "Отчет производства за смену" уже созданные по документу требования-накладные будут автоматически найдены и проставлены в списке требований-накладных.
