Как внести данные о физическом лице

Для того, чтобы внести данные по физическому лицу необходимо открыть в интерфейсе «Бухгалтерский»- меню Кадровый учет - Физические лица, кнопка Добавить.

  1. В появившемся окне в поле Имя внести фамилию, имя и отчество.
  2. На закладке Общие указать дату рождения, место рождения, данные о том, является ли физическое лицо пенсионером. Внести данные о гражданстве, а также данные документа, удостоверяющего личность, ИИН (РНН, СИК).
  3. На закладке Дополнительная информация в поле Карт-счета по кнопке Добавить можно внести данные о наличии платежной карты, на которую в дальнейшем будет перечисляться заработная плата. В поле Контактная информация по кнопке Добавить можно указать адрес, телефон и другую контактную информацию о физическом лице.
  4. По кнопке ОК данные о физическом лице будут записаны.
