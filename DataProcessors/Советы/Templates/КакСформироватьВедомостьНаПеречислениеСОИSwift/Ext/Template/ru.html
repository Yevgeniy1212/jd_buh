Как сформировать ведомость на перечисление социальных отчислений и сформировать Swift-файл для передачи данных в банк

  1. Создайте новый документ СО перечисление в фонды (меню Зарплата – Учет социальных отчислений – СО перечисление в фонды кнопка Добавить).
  2. В поле От введите дату формирования списка на перечисление социальных отчислений.
  3. В поле Структурная единица выберите организацию или структурное подразделение, по которому будут формироваться списки на перечисление.
  4. Поле Подразделение заполняется только в том случае, если необходимо подготовить списки на перечисление по подразделениям. Если необходимо сформировать списки на перечисление по всем сотрудникам организации, то поле Подразделение не заполняется.
  5. В поле Месяц исчисления выберите месяц, за который будет перечисляться задолженность по социальным отчислениям.
  6. В поле Вид платежа должно быть установлено значение Налог (взносы): начислено\уплачено.
  7. В поле Перечислять выбрать вариант заполнения документа.
  8. Для заполнения табличной части нажмите на кнопку Автозаполнение (расположенную на верхней командной панели документа).
  9. После заполнения табличной части документа нажмите на кнопку Записать.
  10. Нажмите на кнопку Действия, выберите команду На основании – Платежное поручение исходящее.

Заполнение документа Платежное поручение исходящее с видом операции: Перечисление социальных отчислений

  1. В поле От введите дату формирования платежного поручения.
  2. В поле Банковский счет  выберите расчетный счет организации, с которого будут перечисляться суммы социальных отчислений.
  3. В поле ГФСС  укажите контрагента, соответствующего Государственному фонду социального страхования.
  4. В поле ГЦВП укажите контрагента, соответствующего Государственному центру по выплате пенсий.
  5. В поле Счет ГЦВП укажите банковский счет ГЦВП  для перечисления социальных отчислений.
  6. В поле Статья ДДС выберите статью движения денежных средств для корректного отражения операции по перечислению социальных отчислений в отчете о движении денег (прямой метод).
  7. На закладке Комиссионное обслуживание  добавлена возможность включения комиссии банка в сумму платежа.
  8. На закладке Печать проверьте реквизиты плательщика, Код назначения платежа, текст назначения платежа.
  9. Для получения печатной формы платежного поручения нажмите на кнопку Печать и выберите значение Платежное поручение.
  10. Для получения печатной формы списка на перечисление обязательных пенсионных взносов нажмите на кнопку Печать и выберите значение Реестр к платежному поручению.
  11. Для того чтобы сформировать Swift-файл нажмите на кнопку Сформировать Swift файл (закладка Основная).
  12. Нажмите на кнопку ОК, для того чтобы сохранить и закрыть документ.
