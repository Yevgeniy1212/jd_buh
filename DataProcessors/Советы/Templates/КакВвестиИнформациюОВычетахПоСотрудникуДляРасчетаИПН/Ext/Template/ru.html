Как ввести информацию о вычетах по сотруднику для расчета индивидуального подоходного налога

Информацию о вычетах можно внести двумя способами: через справочник Физические лица или через справочник Сотрудники.

Через справочник Физические лица:

  1. Откройте в справочнике Физические лица (Кадровый учет – Физические лица) заполненный элемент.
  2. По кнопке Налоги введите информацию о вычетах:
    • На закладке Вычеты добавьте новую строку в табличной части раздела Право на налоговые вычеты (кнопка Добавить в командной панели раздела).
    • В поле Период введите дату, с которой будет предоставляться вычет.
    • Предоставлять личный вычет – установите признак в поле Предоставлять личный вычет, для предоставления вычета в размере 1 МЗП.
    • В поле Количество иждивенцев – укажите количество иждивенцев, для предоставления налогового вычета в размере 1 МЗП на каждого иждивенца.
    Примечание: Поле Количество иждивенцев заполняется в том случае, если данные будут использоваться системой для расчета ИПН за период предшествующий 2007 году.
  3. Добавьте новую строку в табличной части раздела Применение вычетов (кнопка Добавить в командной панели раздела):
    • В поле Период укажите дату, с которой будут применены вычеты.
    • В поле Налогоплательщик укажите организацию, в которой будет предоставляться налоговый вычет физическому лицу при расчете индивидуального подоходного налога.
  4. Сохраните информацию и закройте форму – кнопка OK.

Через справочник Сотрудники:

  1. Откройте в справочнике Сотрудники (Кадровый учет - Сотрудники) заполненный элемент справочника (карточка сотрудника).
  2. На вкладке "Общие" в разделе "Налоговые вычеты, статус налогоплательщика" в поле "Стандартные" нажмите кнопку "...".
    • На закладке Вычеты добавьте новую строку в табличной части раздела Право на налоговые вычеты (кнопка Добавить в командной панели раздела).
    • В поле Период введите дату, с которой будет предоставляться вычет.
    • Предоставлять личный вычет – установите признак в поле Предоставлять личный вычет, для предоставления вычета в размере 1 МЗП.
  3. Добавьте новую строку в табличной части раздела Применение вычетов (кнопка Добавить в командной панели раздела):
    • В поле Период укажите дату, с которой будут применены вычеты.
    • В поле Налогоплательщик укажите организацию, в которой будет предоставляться налоговый вычет физическому лицу при расчете индивидуального подоходного налога.
  4. Сохраните информацию и закройте форму – кнопка OK.
