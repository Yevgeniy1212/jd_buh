Как выполнить операцию приобретения основных средств у поставщика и принять эти основные средства к учету

  1. Введите новый документ Поступление ТМЗ и услуг (Покупка – Поступление ТМЗ и услуг – кнопка Добавить).
  2. Укажите вид операции Покупка (кнопка Операция в командной панели формы документа).
  3. В поле От укажите дату приобретения основных средств.
  4. В поле Организация укажите организацию, в которую поступают основные средства.
  5. В поле Контрагент укажите поставщика основных средств.
  6. В поле Договор укажите договор поставки основных средств.
  7. На закладке Основные средства добавьте новую строку (кнопка Добавить).
  8. В колонке Основное средство выберите приобретаемое основное средство для этого в справочник Основные средства добавьте новый элемент, и заполните поля: Наименование, Изготовитель, Группа учета ОС, Код КОФ.
  9. В колонке Сумма введите сумму приобретенного основного средства.
  10. В колонке Цена укажите цену оборудования.
  11. В колонке % НДС укажите ставку НДС.
  12. В поле Счет учета должно быть установлено значение 2410 «Основные средства».
  13. В поле Вид оборота и Вид поступления ТМЗ (работ, услуг) выберите соответствующий вид оборота и вид поступления ТМЗ (работ, услуг) для корректного учета суммы НДС к зачету, и для правильного отражения операции по приобретению основных средств в декларации по НДС (форма 300.00).
  14. Запишите и проведите документ – кнопка OK.
  15. На основании документа Поступление ТМЗ и услуг введите документ Принятие к учету ОС (кнопка Действия – На основании).
  16. В документе Принятие к учету ОС при этом операция должна быть установлена Приобретенные основные средства (кнопка Операция в командной панели формы документа).
  17. В поле Организация укажите организацию, в которой основное средство принимаются к учету.
  18. На закладке Основные средства, в поле Основное средство, выберите элемент из справочника Основные средства.
  19. В табличной части документа Принятие к учету ОС для автоматического заполнения реквизита Первоначальная стоимость (БУ) нажмите на кнопку Заполнить и выберите значение Для списка ОС (командная панель табличной части документа).
  20. В поле Срок полезного использования, мес. – введите срок полезного использования в месяцах для вычисления суммы амортизации по данному основному средству.
    • В поле Подразделение укажите подразделение, в котором будет учитываться это основное средство.
    • В поле МОЛ укажите материально-ответственное лицо.
    • В поле Порядок погашения – выберите порядок погашения основного средства, если основное средство будет амортизироваться, в поле порядок погашения установите значение – Начисление амортизации.
    • Установите флаг Начислять амортизацию (БУ) – если по принимаемому к учету основному средству необходимо будет начислять амортизацию.
    • В поле Счет учета укажите счет, на котором будет учитываться основное средство (счет 2410).
    • В поле Счет начисления амортизации укажите счет учета амортизации (счет 2420).
    • В поле Способ начисления амортизации укажите способ, для начисления амортизации.
    • Далее следует заполнить поля, которые будут предложены в соответствии с выбранным способом начисления амортизации.
  21. Для Налогового учета ФА необходимо заполнить следующие реквизиты.
    • Если принимаемое основное средство признается фиксированным активом по налоговому учету, установите флаг Признак фиксированного актива.
    • В поле Амортизационная группа НУ выберите группу фиксированных активов по налоговому учету.
    • В поле Порядок погашения стоимости (НУ) выберите порядок погашения стоимости по налоговому учету.
  22. Запишите документ – кнопка Записать.
  23. Напечатайте первичный документ – кнопка Форма ОС-1.
  24. Проведите документ и закройте форму – кнопка OK.
