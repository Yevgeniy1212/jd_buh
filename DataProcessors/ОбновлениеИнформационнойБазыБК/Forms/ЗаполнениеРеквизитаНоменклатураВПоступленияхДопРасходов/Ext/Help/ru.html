Заполнение реквизита "Номенклатура"

Обработка предназначена для заполнения реквизита "Номенклатура" в документах "Поступление доп. расходов" с видом операции "Услуга сторонней организации". Заполнение реквизита необходимо для корректного заполнения документов "Счет-фактура (полученный)", введенных на основании поступлений доп. расходов.

Доступны 3 варианта заполнения поля "Номенклатура" в документах:

  • Установить указанную номенклатуру. При выборе данного варианта выбранная номенклатура будет установлена во всех документах "Поступление доп. расходов" с видом операции "Услуга сторонней организации" и в необходимых документах "Счет-фактура (полученный)".
  • Создать номенклатуру по умолчанию. При выборе данного варианта будет создан элемент справочника "Номенклатура" с наименованием "Услуга сторонней организации (создано при переходе с редакции 2.0)" с типом "Услуга" и установлен во всех документах "Поступление доп. расходов" с видом операции "Услуга сторонней организации" и в необходимых документах "Счет-фактура (полученный)".
  • Создавать номенклатуру по данным поля "Содержание" каждого документа. При выборе данного варианта номенклатура будет создаваться исходя из данных, указанных в поле "Содержание" каждого документа "Поступление доп. расходов". При данном варианте будет создано столько элементов справочника "Номенклатура", сколько будет найдено документов с различным содержанием. Если поле "Содержание" в документе "Поступление доп. расходов" окажется пустым, то будет создана и установлена номенклатура по умолчанию.
