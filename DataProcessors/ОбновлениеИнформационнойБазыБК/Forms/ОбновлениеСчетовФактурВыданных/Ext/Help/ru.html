Обработка счетов-фактур выданных

Обработка предназначена для заполнения новых реквизитов документов "Счет-фактура (выданный)", а также для заполнения нового измерения "Счет-фактура" и нового ресурса "Оборот по реализации в валюте договора" в регистре накопления "Сведения о счетах фактурах выданных".

Обработка выполняет следующие действия:

1. Заполняет новые реквизиты счетов-фактур выданных:

  • "Способ получения",
  • "Вид счета-фактуры",
  • "Дата совершения оборота по реализации".

2. Пересчитывает значение реквизита "Оборот по реализации" из тенге в валюту. Пересчет выполняется только для выданных счетов-фактур, выписанных в валюте начиная с 01.07.2014 г.

3. Заполняет новое измерение и ресурс регистра накопления "Сведения о счетах-фактурах выданных":

  • Измерение "Счет-фактура",
  • Ресурс "Оборот по реализации в валюте договора".
