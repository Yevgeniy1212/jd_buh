Запуск обработок обновления программы

Обработка предназначена для пакетного запуска набора обработок, которые должны запускаться после обновления программы до начала работы. Целесообразно использовать данную обработку в случае, когда последовательно запускать отдельные обработки вручную является неудобным либо неоптимальным.

Список обработок отображается в виде дерева. При пакетном запуске будут выполняться только обработки, отмеченные в списке флажками.
