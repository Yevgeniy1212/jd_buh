Перенос данных из информационных баз 1С:Предприятия 7.7

Обработка "Перенос данных из информационных баз 1С:Предприятия 7.7" - это специализированный Помощник перехода, предназначенный для переноса данных из информационных баз на платформе 1С:Предприятие 7.7.

Помощник перехода, позволяет произвести автоматическую загрузку данных непосредственно из информационной базы на платформе 1С:Предприятие 7.7. Для этого потребуется указать из какой базы производится перенос данных, а также выбрать, какие данные следует переносить – только справочники, или справочники и учетные данные. После загрузки, Помощник проводит созданные документы, а также производит проверку загруженных данных (например, проверяются совпадение итоговых значения оборотно-сальдовой ведомости и т.д.). В случае возникновения каких-либо ошибок, Помощник позволяет сформировать отчет об ошибках с указанием этапа, на котором они возникли.

Помощник перехода позволяет также загружать данные из файла выгрузки в "1С:Бухгалтерию 8 для Казахстана".
