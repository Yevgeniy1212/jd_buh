Обработка «Копирование табличных частей документов»

Обработка позволяет копировать данные из табличной части одного документа в табличную часть другого (например, заполнить платежную ведомость на выплату заработной платы работниками и суммами из документа по начислению премий), задавая произвольное соответствие реквизитов и условий отбора.

В формах документах, поддерживающих использовании данной обработки, на командной панели над табличным полем, связанного с табличной частью документа, имеется кнопка "Заполнить", при нажатии на которую открывается меню с двумя пунктами для вызова обработки в разных режимах. При выборе пункта "Заполнить по документу" сначала происходит удаление всех строк табличной части документа, после чего вызывается обработка, при выборе пункта "Добавить из документа" - удаление существующих строк табличной части документа не происходит, данные, сгенерированные обработкой добавляются к уже имеющимся.

Окно обработки содержит:

  • Поле выбора Исходный документ служит для указания документа, из которого будут копироваться данные в целевой документ.
  • Список Табличная часть предназначен для выбора табличной части исходного документа, из которой строки будут копироваться в целевой документ.
  • Табличное поле Соответствие реквизитов предназначено для установки соответствия между реквизитами табличных частей целевого и исходного документов. При выборе исходного документа система автоматически формирует соответствие реквизитов, имеющих одинаковое имя и тип значения. Соответствие может быть также установлен вручную. Для этого надо выбрать реквизит табличной части исходного документа в колонке "Из реквизита" из выпадающего списка, появляющегося при начале редактирования. Выпадающий список содержит перечень реквизитов табличной части исходного документа, тип значения которых совместим с типом целевого реквизита. Кроме того, в конце списка содержится специальный элемент, обозначаемый многоточием, выбор которого означает, что в данный целевой реквизит необходимо будет установить значение, указанное в колонке "Прочее значение" для всех добавляемых строк. При копировании строк табличной части будут заполняться только целевые реквизиты, отмеченные флажком.
  • Кнопка Автозаполнение восстанавливает соответствие реквизитов табличных частей документов по умолчанию.
  • Табличное поле Отбор позволяет задать условия выбора строк из исходного документа. В результате отбора в целевой документ будут перенесены только строки удовлетворяющие всем критериям, заданным на закладке Отбор.

После задания соответствия реквизитов и условий отбора строк следует нажать на кнопку ОК, в результате чего окно обработки будет закрыто, и строки, удовлетворяющие указанным критериям отбора, будут перенесены из выбранной табличной части исходного документа в текущий документ, из которого была вызвана обработка, в соответствии с указанными правилами сопоставления реквизитов.

Сделанные настройки правил сопоставления реквизитов и условий отбора могут быть сохранены и использованы в дальнейшем.
