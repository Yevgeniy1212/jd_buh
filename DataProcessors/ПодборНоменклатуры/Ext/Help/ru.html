Обработка «Подбор номенклатуры»

Обработка предназначена для подбора позиций номенклатуры в табличную часть документа. Вызывается обработка при нажатии на кнопку «Подбор» из документов, имеющих табличную часть «ТМЗ» или «Услуги». При этом открывается справочник «Номенклатура» в режиме подбора. Если обработка вызывается из табличной части «ОС», то открывается справочник «Основные средства» в режиме подбора.

В поле «Подбор» выбирается вариант подбора позиций номенклатуры, от которого зависит список позиций и показываемые графы в окне подбора. Цены и остатки позиций номенклатуры показываются на дату и время документа. Эта информация отображается в поле «Остатки и цены на:». Дату и время, на которые будут отображены остатки и цены, можно изменить. При этом произойдет автоматическое переформирование списка позиций номенклатуры, в соответствии с новыми ценами и остатками. На список позиций номенклатуры влияют и организация, и склад, указанные в документе. Изменить эту информацию в окне подбора нельзя. Цены номенклатуры указаны с учетом типа цены в документе (форма «Цены и валюта»). Если в документе тип цены не указан, в окне подбора позиций номенклатуры цена тоже не будет указана.

Список вариантов подбора:

  • По справочнику;
  • По остаткам номенклатуры;
  • Остатки и цены номенклатуры;
  • По остаткам и плановой себестоимости;
  • По ценам номенклатуры;
  • По услугам;
  • По ценам услуг.

При подборе можно выбрать определенный режим запроса позиций номенклатуры. Для этого предназначены признаки Запрашивать, Количество и Цена. При установке одного или обоих признаков система будет выдавать пользователю запрос заданных параметров позиции номенклатуры. Если ни один из признаков не установлен, то никакого запроса выдаваться не будет, а при повторном выборе позиции номенклатуры количество будет увеличиваться на 1.

Признак Показывать только номенклатуру, имеющую остатки предназначен для отбора списка позиций номенклатуры с ненулевым остатком. Признак доступен только для вариантов отбора по остаткам номенклатуры, по остаткам и ценам, по остаткам и плановой себестоимости.

Вариант подбора «По справочнику» показывает весь список позиций номенклатуры в том виде, как он зафиксирован в справочнике «Номенклатура». Такой вариант подбора в основном используется в случае, когда наличие остатков и цен товаров не имеет значения.

Вариант подбора «По остаткам номенклатуры» показывает весь список позиций номенклатуры, но уже с указанием остатков на складе. При таком варианте подбора становится доступным признак Показывать только номенклатуру, имеющую остатки. Если он установлен, то показываются только те позиции номенклатуры, которые имеются на указанном в документе складе.

Вариант подбора «Остатки и цены номенклатуры» показывает весь список позиций номенклатуры с указанием остатков и цен. При таком варианте подбора становится доступным признак Показывать только номенклатуру, имеющую остатки. Если признак установлен, то показываются только те позиции номенклатуры, которые имеются на складе. Цена указывается только в том случае, если в регистре сведений «Цены номенклатуры» для позиции номенклатуры зафиксирована цена. Тип цены зависит от типа цен в самом документе. Т.е. если в документе в форме «Цены и валюта» указан тип цены «Приходная», то и при подборе номенклатуры будет указана приходная цена позиций номенклатуры. Если в регистре сведений для позиции номенклатуры нет цены по указанному типу цен, то при подборе значение цены будет пустым.

Вариант подбора «Остатки и плановая себестоимость» показывает весь список позиций номенклатуры с указанием остатков и плановой себестоимости. При таком варианте подбора становится доступным признак Показывать только номенклатуру, имеющую остатки. Если признак установлен, то показываются только те позиции номенклатуры, которые имеются на складе. Цена указывается только в том случае, если в регистре сведений «Цены номенклатуры» для позиции номенклатуры зафиксирована цена с типом цен «Плановая». Т.е. если в регистре сведений «Цены номенклатуры» для позиции номенклатуры есть запись с типом цены «Плановая», то при подборе номенклатуры будет указана плановая цена позиций номенклатуры. Если в регистре сведений для позиции номенклатуры нет цены по указанному типу цен, то при подборе значение цены будет пустым.

Вариант подбора «Цены номенклатуры» показывает весь список позиций номенклатуры с указанием цены. Цена указывается только в том случае, если в регистре сведений «Цены номенклатуры» для позиции номенклатуры зафиксирована цена. Тип цены зависит от типа цен в самом документе. Т.е. если в документе в форме «Цены и валюта» указан тип цены «Приходная», то и при подборе номенклатуры будет указана приходная цена позиций номенклатуры.

Вариант подбора «Услуги» возможен в том случае, если подбор номенклатуры осуществляется в табличную часть «Услуги». При таком варианте подбора показывается весь список услуг, т.е. из справочника «Номенклатура» выбираются все позиции с установленным признаком Услуга.

Вариант подбора «Цены услуг» показывает весь список услуг с указанием цены, т.е. из справочника «Номенклатура» выбираются все позиции с установленным признаком Услуга. Цены указываются в том случае, если в регистре сведений «Цены номенклатуры» для позиции номенклатуры зафиксирована цена с указанным в документе типом цен. Если в регистре сведений для позиции номенклатуры нет цены по указанному типу цен, то при подборе значение цены будет пустым. Такой вариант подбора возможен в том случае, если подбор позиций номенклатуры осуществляется в табличную часть «Услуги».

Выбранные позиции номенклатуры помещаются в таблицу «Подобранные позиции». При необходимости в таблице можно изменить и количество, и цену номенклатуры.

Кнопка «Перенести в документ» предназначена для переноса списка позиций номенклатуры из табличной части «Подобранные позиции» в табличную часть документа.
