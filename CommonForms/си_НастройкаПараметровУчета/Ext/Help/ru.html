Настройка параметров учета спецодежды и инвентаря

Форма предназначена для настройки параметров конфигурации "Учет спецодежды и инвентаря".

Параметры вкладки "Аналитический учет":

Вести учет спецодежды и спецоснастки по характеристикам - включает учет спецодежды и инвентаря по характеристикам

Счет учета в эксплуатации - указанный счет будет использоваться в проводках для материалов в эксплуатации

Счет учета на складе - указанный счет будет использоваться в проводках для материалов на складе

Учет спецодежды и инвентаря по физическим лицам - если установлено, ведение учета происходит по справочнику "Физические лица", иначе по справочнику "Сотрудники"

Параметры вкладки "Учетная политика":

Ведется контроль спецодежды и инвентаря по нормам - включает контроль норм при учете спецодежды и инвентаря

Способ расчета норм выдачи - определяет правила проверки норм обеспечения спецодеждой и инвентарем по должностям. Если установлено "Максимальное значение норм", то берется максимально разрешенное количество среди должностей работника. Если "Сумма норм", то для определения максимально разрешенного количества, нормы по всем должностям суммируются.

Порядок начисления амортизации - определяет месяц начисления амортизации для материалов.

Амортизировать материалы на складе в месяц возврата - если установлено, то материалы, возвращенные на склад амортизируются в месяц возврата, вне зависимости от признака "Приостанавливать амортизацию на время возврата" в назначении использования материала.

Параметры вкладки "Настройки СЛК":

Расположение сервера - определяет местоположение лицензионного ключа

Имя компьютера или IP-адрес - IP-адрес, либо DNS-имя сервера лицензий

Сетевой порт - порт для подключения к серверу лицензии

Команда "Установить соединение" - производит подключение к серверу лицензий по указанным параметрам
