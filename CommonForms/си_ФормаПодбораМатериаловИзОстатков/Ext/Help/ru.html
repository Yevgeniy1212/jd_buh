Форма подбора остатков материалов

Форма предназначена для подбора остатков материалов, находящихся в эксплуатации, либо на складе.

Состоит из трех основных элементов:

  • Список материалов в остатках - отображает список материалов в эксплуатации, либо на складе, доступных к использованию;

  • Параметры отбора материалов - задаются параметры отбора в списке материалов в остатках;

  • Список выбранных материалов - список материалов, который будет загружен в вызывающую форму

Команды управления списком выбора материалов:

  • Выбрать - производит загрузку выбранного материала в список "Выбранные";

  • Отбор по текущему - производит отбор по текущему полю списка. Отбор возможно произвести по:

    • Номенклатура;

    • Склад;

    • Сотрудник;

    • Назначение использования;

    • Документ передачи.

    при нажатии кнопки отбора соответствующее поле области параметров отбора заполнится, произведется отбор в списке материалов. Если поле подбора очистить, список материалов снимет отбор по этому полю;

  • Очистить параметры отбора - производит очистку полей области параметров отбора, снимает отбор в списке материалов;

  • Упорядочить по возрастанию - упорядочивает список материалов по возрастанию по текущей колонке;

  • Упорядочить по убыванию - упорядочивает список материалов по убыванию по текущей колонке;

  • Выделить все - выделяет все строки списка материалов (параметры отбора учитываются);

  • Снять выделение - отменяет выделение строк;

  • Обновить список - обновляет данные списка материалов;
  • Группировка - устанавливает или снимает группировку в списке материалов;
  • Группировка по текущей колонке - устанавливает группировку списка материалов по текущей колонке, группируемая колонка становится лидирующей в списке.

 

Примечание: Доступность некоторых элементов формы зависит от типа формы подбора (на складе/в эксплуатации).

После выбора всех необходимых материалов, необходимо нажать кнопку "Загрузить выбранные" для загрузки материалов из списка "Выбранные" в вызывающую форму.
