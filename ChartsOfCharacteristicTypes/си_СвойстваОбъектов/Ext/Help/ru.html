План видов характеристик "УСИ: Свойства объектов"

Для каждого объекта могут быть назначены значения видов свойств различного типа. Значения видов свойств заносятся с помощью обработки "УСИ: Значения свойств объектов".
Для свойств объектов могут быть назначены различные типы значения свойств. 

Заполнение реквизитов:

  • Наименование - наименование свойств объекта.
  • Тип значений - тип значения свойств объекта.


В табличной части "Значения свойств объекта" вводится список возможных значений для данного вида свойства.
